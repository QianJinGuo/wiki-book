# 百度提出 Unlimited OCR：用 Reference Sliding Window Attention 实现长文档 OCR 常量 KV Cache

## 核心要点

1. 百度提出 Unlimited OCR，用 Reference Sliding Window Attention（RSWA）替代 DeepSeek-OCR 解码器中的标准注意力
2. RSWA 使长文档 OCR 的 KV cache 保持常量，解决了长文档推理的显存瓶颈
3. 在 OmniDocBench 上总体分数达到 93.23
