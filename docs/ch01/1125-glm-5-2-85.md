# 实测GLM-5.2百万上下文：85页世界杯前瞻

花叔对智谱 GLM-5.2 百万上下文窗口的实测体验，生成了 85 页的世界杯前瞻报告。

## 背景

在 Anthropic Fable 5 / Mythos 5 因美国商务部国家安全要求被关停的当天（2026-06-13），智谱发布公告并开放 GLM-5.2 百万上下文能力。作者在两种模型交替事件背景下对 GLM-5.2 进行了实测。

## 实测结果

- 使用 GLM-5.2 的百万上下文窗口，生成了一份 **85 页的世界杯前瞻报告**
- 展示了百万上下文在实际场景中的应用能力（体育赛事分析这类需要大量历史数据的场景）
