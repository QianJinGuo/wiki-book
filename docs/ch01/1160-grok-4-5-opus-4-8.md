# Grok 4.5：Opus 4.8 级能力，四分之一价格

> **Grok 4.5** 是 xAI 于 2026 年 7 月发布的最新大语言模型，定位为 Opus 4.8 级别的能力，定价仅为四分之一。模型参数量为 1.5T（推测），上下文窗口 500K tokens，支持文本和图像输入，文本输出。

## 关键参数

- 上下文窗口：500K tokens
- 模态：文本 + 图像输入，文本输出
- 定价：输入 $2/百万 tokens，输出 $6/百万 tokens
- 知识截止时间：2024 年 11 月（与 Grok 3/4 系列相同）

## 与 Grok 4.3 对比

Grok 4.5 的价格是 Grok 4.3 的 2.4 倍（输出侧），上下文窗口从 Grok 4.3 的 1M tokens 缩减至 500K。这反映了 1.5T 大参数在推理成本上的直接代价。

## 使用方式

模型通过 Grok Build、API 和 xAI Console 三个入口同步开放。开发者可通过 FastAPI/SpaceXAI Console 开通 API key，模型名为 `grok-4.5`。
