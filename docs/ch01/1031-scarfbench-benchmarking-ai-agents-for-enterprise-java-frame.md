# ScarfBench: Benchmarking AI Agents for Enterprise Java Framework Migration

ScarfBench is an IBM Research benchmark for evaluating AI agents on enterprise Java framework migration tasks — specifically migrating from older frameworks (Jakarta EE, Spring) to modern ones. The benchmark covers real-world migration scenarios including dependency injection refactoring, XML-to-annotation configuration migration, persistence layer updates, and security framework upgrades. Each task has a reference solution, test suite, and scoring rubric. The benchmark reveals that current AI agents (Claude, GPT-4, open-source code LLMs) struggle with multi-file coordinated changes, understand partial migrations but miss cascading effects, and produce compilation errors in 40-60% of cases. Gaps include lack of build system awareness, insufficient test generation, and poor handling of framework-specific idioms.
