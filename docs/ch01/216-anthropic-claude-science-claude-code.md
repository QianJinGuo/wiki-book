这两天，Anthropic有点忙。 发布Sonnet 5，Fable 5 宣布解禁，还低调推出了一款旗舰产品：**Claude Science。**

Claude Science 是面向科研人员的 AI 平台，主打“一个界面搞定科研工作流”。

不是新的模型，但是重要性和模型相当。

任何一个写过学术论文的人，都体会过阅读整理文献的那种绝望。。

上百篇论文散在 PubMed、OpenAlex 和各家期刊里，光是判断哪些值得读、这个方向这两年到底在往哪走，之前往少了说也得半个月。还有科研党天天都离不开的跑数据、画图、写稿、校验结果……想想就头大。

Claude Science，瞄准的正是这段最熬人的过程。

**没有特殊权限，没有内测限制。**

Claude Science 支持 macOS 和 Linux 系统，Pro、Max、Team、Enterprise用户现在就能用！

生物学家们这次终于也能大大方方了——之前 Fable 5 ，他们连发个 Hi 都发不出去。
