# 京东 Oxygen AIIC：工业级百亿商品大模型商品中心

> 京东 Oxygen AI Item Center（arXiv: 2606.28070），以 LLMs/VLMs 为核心的工业级商品知识生产与服务平台，覆盖万级类目、支撑亿级日更新、沉淀千亿级商品知识资产。

## 四大技术支柱

### 1. 人机协同本体演进
- **专家定义骨架（自顶向下）**：类目/属性键/属性值/场景标签
- **算法驱动生长（自底向上）**：知识发现 → 知识融合 → 知识验证
- 百万级本体，刻画维度 +44%，本体规模 +64.5%，用户流量覆盖 80.4%

### 2. S²D 架构（语义搜索—判别）
Semantic Search-Then-Discrimination 将"自由生成"转化为"有限候选内筛选"，本体演化无需重训模型。

**三重高吞吐优化**：SKU 计算复用 + 属性相关性探测 + 缓存共享前缀（SPU 下 85% 相同）+ 异步异构流水 → 吞吐效率 **10× 提升**

结果：Precision 92% / Recall 78.3%，SKU 属性数 1.5×，知识资产 3.35×（千亿级）。

### 3. 商品理解大模型（自演进）
- **多任务基座**：知识抽取+识别多任务指令微调
- **增量学习**：LoRAM（幅值优化）+ GROLE（GRPO 动态路由）
- **InstEmb 推理型表征**：隐式 CoT 蒸馏，Latent CoT 标记一次前向出向量
- **FANoise 谱结构噪声注入**：SVD 分解按谱分布加噪保护弱信号
- **模型自演进**：数据评估→分析→合成→筛选四阶段闭环

最终：端到端 **Precision 94.2% / Recall 82.8%**，新增本体周期从 30+ 天缩短至 2 周。

### 4. 商品总线（Item Tunnel）
分级时效（离线/近线/实时）+ 版本一致性 + 存算效率 + 统一服务，支撑每日亿级更新。
