# Introducing Claude Platform on AWS: Anthropic’s native platform, through your AWS account
Today, we’re excited to announce the general availability of Claude Platform on AWS. Claude Platform on AWS is a new service that gives customers direct access to Anthropic’s native Claude Platform experience through their AWS account, with no separate credentials, contracts, or billing relationships required. AWS is the first cloud provider to offer access to the native Claude Platform experience.
In this post, we explore how Claude Platform on AWS works and how you can start using it today.

## 深度分析

Claude Platform on AWS 的发布标志着 Anthropic 云服务战略的重大转变。通过将原生 Claude Platform 体验直接嵌入 AWS 账户，Anthropic 消除了此前阻碍企业采用的关键障碍——多账号管理、割裂的计费体系和不一致的认证机制。

这一合作的深远意义在于：AWS 作为全球最大的公有云提供商，成为 Anthropic 首个原生集成合作伙伴。这意味着企业客户可以在熟悉的 AWS 管理控制台中直接调用 Claude 的能力，实现与现有 AWS 资源（IAM、S3、Lambda 等）的无缝协作。

从竞争格局看，此举使 Anthropic 在企业级 AI 落地层面与 Microsoft Azure OpenAI Service 形成直接对标，但通过"无感计费"（纳入 AWS 账单体系）降低了客户的采购摩擦。

## 实践启示

1. **简化企业采购流程**：对于已有 AWS 成熟使用的企业，无需新增供应商谈判周期，可直接通过 AWS Marketplace 或 Console 启用 Claude Platform，降低采购阻力。

2. **统一身份与权限管理**：借助 AWS IAM 体系，企业可实现细粒度的访问控制，无需维护独立的 Anthropic 账号体系，减少了权限孤岛和潜在的安全风险。

3. **成本归一化**：Claude 消费可纳入现有 AWS 消费报告和预算体系，便于 CFO 进行统一成本管控和分摊。

4. **混合云场景适配**：适合已在 AWS 上构建核心业务的客户，无需跨云协调即可在 AWS 环境内完成 AI 能力的集成与部署。
