- 当然，官方的案例和官方的声称毕竟是官方的，真实水平如何还得我自己测了才行。于是我用 Hy3 跑了些测试，甚至还让 Claude Code 当裁判，拉它和 GLM 模型进行了一场 PK（胆大如我依然敢用 cc）。
