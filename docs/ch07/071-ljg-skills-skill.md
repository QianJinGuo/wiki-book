# 李继刚 ljg Skills 系列（四）：表达写作类 Skill

## 摘要

李继刚 6.1k 星 GitHub 项目 Skills 系列第四篇，聚焦"输出"环节——如何将想清楚的内容有效地表达给别人听。介绍了三个核心 Skill：ljg-plain（拆术语）、ljg-writes（聚焦观点）、ljg-invest（秩序创造机器视角）。

## 三个表达类 Skill

### ljg-plain：拆翻译腔

**目标**：把术语和翻译腔拆掉，让表达回归本质。
**适用场景**：写出来像 AI 说明书的用户，需要让文字更自然、更易懂。

### ljg-writes：聚焦观点

**目标**：对准一个观点下刀，不跑题、不散漫。
**适用场景**：需要在有限篇幅内传递清晰的核心信息，避免"什么都说了等于什么都没说"。

### ljg-invest：秩序创造机器

**目标**：用"秩序创造机器"的视角看项目，从混乱中找出规律和结构。
**适用场景**：分析复杂项目时，需要一个框架来组织观察结果。

## 实践启示

- 如果你经常写出来像 AI 说明书，先从 ljg-plain 开始
- 三者的递进关系：先去掉不好的表达 → 再聚焦要说什么 → 最后用框架组织
