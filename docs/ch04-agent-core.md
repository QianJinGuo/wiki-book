## 导读

这是全书最厚的一章——因为 Agent 是 2024-2026 年 AI 领域最核心的范式转移。

Karpathy 把它称为"从 Vibe Coding 到 Agentic Engineering"：不是让 AI 写代码，而是让 AI 自主完成整个工程任务。这个转变的深度远超大多数人的想象。

本章从"什么是 Agent"这个基本问题开始，经过 ReAct、Plan-and-Execute、Reflection 等经典架构，一路走到 Claude Code 的 Dynamic Workflows、OpenClaw 的多 Agent 团队、以及 AgentRun 的 A2A 协议。你会看到 17 种 Agent 架构的完整演化史，理解为什么"Harness 决定上限，模型决定下限"。

这是理解整本书后续内容的核心——Harness、记忆、技能、多 Agent、编码 Agent，全部建立在本章的概念基础之上。
