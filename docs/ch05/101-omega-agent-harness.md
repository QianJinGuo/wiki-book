# OMEGA: 面向多机器人协作的具身Agent Harness

OMEGA（Orchestration system for Multiple Embodied Generalist Agents）是一套面向真实世界多机器人协作的embodied agent harness，由中国科学技术大学张举勇教授团队联合上海人工智能实验室共同完成。

OMEGA覆盖多机器人协作的六个关键能力层：任务感知与拆解、动态任务分配、并行执行控制、状态同步与等待协调、异常检测与重调度、以及技能沉淀与复用。系统支持异构机器人（不同形态、不同能力集）在同一框架下协作完成长程任务。

与现有Agent orchestration方案（[Agent Orchestration Multi Agent Systems](../ch04/482-agent-orchestration.html)、[Coze 3 Multimagent Team Orchestration Wangheige](../ch03/045-agent.html)）不同，OMEGA解决的是物理世界中的多机器人协调问题——包括运动规划冲突、物体交接时序、传感器信息融合和物理约束下的任务重规划，这些是纯软件agent orchestration不会遇到的独特挑战。
