# 企业 AI 的非技术困境：本体驱动 Agent 与知识治理

## 摘要

DataFunTalk 圆桌讨论实录，围绕本体建模、知识治理、企业 AI 转型与组织变革等核心议题，探讨企业 AI 落地中"非技术"层面的挑战。

## 关键议题

### 本体是技术问题还是管理问题？
- 本体建模不仅是技术工作，更是业务梳理和组织对齐
- 需要业务与技术团队的持续协作

### 知识治理的落地路径
- 知识治理是 AI 可用的前提，而非 AI 的结果
- 从无序到有序的渐进式治理策略

### Agent 驱动的企业 AI 转型
- 本体作为 Agent 的知识底座
- 治理->可用->智能的渐进路线
