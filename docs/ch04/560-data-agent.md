### 核心观点
1. # 什么是数据智能体Data Agent--数据智能体-火山引擎
数据智能体 Data Agent（后文简称 Data Agent）是火山引擎依托大模型技术，推出的新一代企业级 AI 数据专家："Data"代表其深度理解和运用企业数据资产的能力，"Agent"体现其作为企业的智能代理，能够像专家一样主动思考、分析和行动。
2. Data Agent能够深度理解和运用企业数据资产，主动思考、洞察、分析与行动，持续释放数据价值，让每个组织都能构建自主进化的数据大脑。
3. ## Data Agent可以做什么
Data Agent 作为企业级数据智能体，以多场景覆盖能力为基础，通过主动思考挖掘数据关联、智能洞察业务趋势、精准分析问题本质并驱动自动执行，全天候高效释放数据价值，构建起数据资源与业务行动之间的智能转化桥梁，实现从数据洞察到业务落地的全周期闭环赋能。
4. ## 企业多场景覆盖
Data Agent 凭借领先的数据驱动技术，构建了多元化的应用场景体系，目前已在智能分析、智能会话、智能营销三大核心领域展现出显著的应用价值与创新活力。
5. **智能分析**：基于AI深度思考与大数据分析的专家顾问。
