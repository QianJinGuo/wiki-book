Sebastian Cuadros 基于 LinkedIn 5306 人数据筛选出 1680 名 Anthropic 工程师的画像分析：不是"博士实验室"，而是高速扩张的基础设施公司。

## 一句话

**Anthropic 工程团队 = 12 年经验资深工程师 + 10 个月任职中位数 + 40% Infra 背景 + 13.7% 博士 + 80% MoTS 平级 + Google 系第一大来源。**

## 核心数据

| 指标 | 数值 |
|------|------|
| 工程师总数 | 1,680（从 5,306 LinkedIn 员工筛选） |
| 任职中位数 | 10 个月 |

## 典型工程师画像

**资深型**：12 年工作经验，10 个月任职，Google/Meta/Amazon 背景，后端/分布式系统/数据库

**年轻型**（<6 年，172 人）：MIT+IOI 银牌+Codeforces 2900+，博士比例 19%，通过实习管道/量化交易/AI 对齐奖学金进入

## Infra > Research

- **40% Infra 背景**（后端/分布式/数据库/安全各约 20%）
- **3.3% RL 背景**
- 技能分布：Python 585 / Java 566 / C++ 443 / JS 376 / SQL 302 / Linux 230 / 分布式 189 / AWS 154

## 薪酬亮点

- **TPU Kernel Engineer**：28-85 万美元（最高薪）
- **Claude Code 模型性能 SWE**：40.5-48.5 万美元
- **集群基础设施工程师（伦敦）**：32.5-48.5 万英镑（≈43.9-65.5 万美元）
- **数据中心 OFE 战略寻源**：29-36.5 万美元

## 人才来源 Top 10

Google 405 / Meta 273 / Amazon 197 / Microsoft 171 / Stripe 124 / Apple 87 / Stanford 68 / DeepMind 62 / Airbnb 51 / OpenAI 48

**关键发现**：最大人才管道是 Google，不是 AI 实验室。约 94 人直接从前沿 AI 实验室跳槽。

## 学校来源 Top 5

Stanford 144 / Berkeley 118 / MIT 80 / CMU 73 / Harvard 42。前四所占工程组织约 1/4。

## 职级压平

80% 共用"Member of Technical Staff"头衔。前 Instagram CTO、前 Adept 创始人、斯坦福教授都只是 MoTS。

## 岗位结构断裂

- 顶级科技公司 SWE 岗位较一年前增长约 20%
- AI 工程职位增长 50%-100%
- 但 AI 暴露度高的职业中，早期职业阶段就业明显下降
- "薪资可能更高，但岗位更稀缺"
