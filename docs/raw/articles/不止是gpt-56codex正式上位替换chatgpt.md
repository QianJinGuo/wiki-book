# 不止是GPT-5.6！Codex正式上位替换ChatGPT

这两天新模型跟不要钱一样往外冒。7月8日Grok 4.5，7月9日更热闹，GPT-5.6和Meta的Muse Spark 1.1挑了同一天发。我人在纽约，头一回不用熬夜蹲发布，白天在街头逛着逛着就能被新模型的消息chuachua往脸上糊的。这大概是我这辈子离AI前沿最近的一次。

不过要说这两天最让我觉得好玩，不是这三个模型里的任何一个。是我打开Codex的时候，它弹了个窗。

没看错，**Codex桌面app直接变成了新的ChatGPT app** 。原来那个聊天界面的ChatGPT，改名叫「ChatGPT Classic」。它还挺贴心，问你要不要保留Codex的应用图标，那语气像公司改组时新领导安抚老员工：工位还是你的，就是门牌换了。

改的只是Mac和Windows的桌面端，手机和网页上的ChatGPT一个像素没变。但这事越想越有意思。一个给程序员写代码的agent工具，成了ChatGPT在桌面上的主力；那个定义了整个AI时代的聊天框，成了经典版。

要理解OpenAI为什么这么做，得先看这三个模型。有意思的是，看完你会发现，它们和这次改名说的是同一件事。

## Grok 4.5：便宜到分数不重要

先注意一个有趣的细节：发布方不叫xAI了，叫SpaceXAI（这读起来也太别扭了。xAI今年2月并入SpaceX，6月SpaceX完成史上最大IPO（募资750亿美元，上市估值1.77万亿），紧接着甩出600亿美元宣布收购Cursor。所以Grok 4.5身上挂着两个第一：SpaceXAI改名后的第一个模型，也是官方口径「与Cursor团队共同训练」的第一个成果。

对于在美国的这几家不差钱的模型公司来说，他们缺的不是算力，是带反馈的真实数据。而Cursor手里正是全世界最大的一批：几百万开发者每天在里面写代码、改代码、骂模型、回滚重来。哪个补全被接受了，哪个agent方案被人推翻了，全是别家拿不到的信号。这批数据开始喂进Grok，意味着**Grok的coding能力从这一代起真的可以认真看一看了** 。我以前对Grok写代码的印象基本是凑数选手，这次得重新校准。SpaceXAI的官宣推文也把这点当成第一卖点，直接说他们是专为编程和agent训练，并且是与Cursor共同训练。两个原本被A社和OpenAI分别打得找不着北的loser，合作起来还是挺优势互补的。

分数确实够到了第一梯队的门槛。Artificial Analysis的智能指数54分，168个模型里排第4，前面只剩Fable 5、Opus 4.8和GPT-5.5；其中agentic工具调用这一项是全榜第一。AA独立复测的Terminal-Bench 2.1上它拿81.6%，排在这三家后面，差距3个点以内。但纯写代码还是露怯：SWE Bench Pro只有64.7%（官方数据），Fable 5是80.3%（Anthropic自报口径），差15个点不是误差，是断层。这不奇怪，收购6月中旬才官宣，数据飞轮刚接上水管，红利恐怕要到下一代才兑现。这一段是我的推测，我们可以到时候回来验证。

它真正的杀手锏算是价格...定价$2/$6每百万token，甚至可以和几家中国头部模型竞争了。而且AA统计过，同样解一道SWE Bench Pro的题，它平均用1.6万个输出token，Opus 4.8要6.7万个。**干同一件事，花销差4倍** 。The Decoder的标题把话说透了：便宜到benchmark上的差距可能都不重要了。Cursor CEO Michael Truell站台说这是「我们做过的最大一步提升」。当然，他刚以600亿把公司卖给人家。换我收了这笔钱，我也觉得是最大一步提升。

所以它适合什么？批量跑agent工作流、成本敏感、有人兜底验收的活，它是眼下性价比最高的选择。拿它查资料写东西？54%的幻觉率，还是先算了...当然，大多数人拿Grok写的也不是什么正经玩意儿，可能幻觉是个好的feature？

## Muse Spark 1.1：不砌墙，想当包工头

Meta的模型，Alexandr Wang接手后第一个开放API的作品。它最大的新闻点不在能力，在商业模式：**这是Meta历史上第一次对模型访问收费** 。做了三年开源Llama的Meta，这次不开源了，彭博社直接把「Meta starts charging」做成标题。扎克伯格为了宣布这件事，时隔3年回X发了帖。跑到马斯克的地盘上给自家模型吆喝，这本身又成了一条新闻，光第一条帖子就840万浏览。上一次这两人靠这么近，还是约笼斗那回。这次小扎没练柔术，带了张benchmark表。

分数很偏科，但偏得有性格。扎克伯格帖子里贴的就是官方对比表（注意对比对象是GPT-5.5，不是同天才发的5.6）。工具编排类是真领先：MCP Atlas 88.1，Opus 4.8才82.2；带工具的Humanity's Last Exam 62.1，也是第一；金融、医疗这类专业agent评测同样压过Opus 4.8。但纯写代码立刻掉下来，SWE-Bench Pro 61.5，比Opus 4.8低了近8个点；长上下文检索MRCR被GPT-5.5甩开20个点。

看出来了吗？它压根没想跟别人卷写代码。**它想当的是包工头：自己不砌墙，负责给一群子agent派活** 。官方专门强调它训练过给并行子agent分发任务，上下文1M（Grok 4.5的两倍）还带自动压缩管理，定价$1.25/$4.25，输入价只有GPT-5.6旗舰的四分之一。

社区反应，HN上最多的声音是「便宜到不敢信」。还有人给了个我很认同的评价：它把agent推向了「正常软件的习惯」，便宜的API、工具调用、子agent、计算机使用，全是工程化的路数。Simon Willison拿到几天预览权限，跑了他那个经典的鹈鹕骑自行车SVG测试，结论是「有点方正但可认」。总体共识是Meta还在追赶Anthropic和OpenAI，但这次追的姿势对了。

想想小扎这一年多砸出去的钱：上百亿美元把Alexandr Wang连人带公司请进门，九位数的薪酬包挖遍各家实验室，Llama 4还折腾得灰头土脸。Muse Spark 1.1不是最强的模型，但它有清晰的定位、有真领先的单项、有让HN喊「不敢信」的定价。**砸了这么多钱，小扎终于算是上桌了** 。

如果你在搭多agent流水线，让一个便宜、能管1M上下文、专门练过调度的模型当中枢，把贵的模型留给关键工序，这个组合蛮合理。可惜API目前只对美国开发者开放预览，国内朋友先看着。

## GPT-5.6：一家三口，正面刚Fable 5

OpenAI一口气发了三档：旗舰Sol、日常款Terra、快速款Luna，外加一个高算力的Sol Ultra模式。太阳、大地、月亮，从GPT-4o、o3那套乱码一步跨到拉丁文，看得出IPO前连命名都做了上市美容。这次发布还有个时代注脚：因为特朗普6月初签的AI网络安全行政令，强模型公开前要送审30天，所以GPT-5.6在6月26日先给了约20家政府审批过的「信任伙伴」，今天才对公众放开。前沿模型发布要政府批准，一年前还是科幻情节，现在是流程。

这次OpenAI的对比表有个耐人寻味的细节：他们把Anthropic的两个版本都搬上来了，Fable 5和Mythos 5，后者是同一个模型的无限制版，只开放给审批过的机构。头条数字打的是agentic coding：Terminal-Bench 2.1上Sol拿88.8%，Ultra模式91.9%，压过Mythos 5的88.0%。Ultra不是更大的模型，是个高算力模式，默认并行指挥4个子agent跑长任务，Codex里Plus订阅就能用。

诚实的地方OpenAI也照登了。SWE-Bench Pro，Sol只有64.6%，Fable 5是80.3%（Anthropic自报口径），15个点的断层就挂在自家发布页上。登完还顺手补了一句：这个benchmark约30%的题目本身有毛病。Simon Willison看完直接点破，这是在给64.6%找台阶。更掉价的是FrontierMath最难档，Fable 5拿87.8%，Sol只有65.9%，连自家上代的72.5%都没打过。顺便说个巧合：Sol这个64.6%，和前一天Grok 4.5的64.7%几乎重合，两家在Fable 5面前坐的是同一条板凳。

第三方的数怎么说？Artificial Analysis的复测里（AA给OpenAI做过发布前评测支持，「独立」两个字打个八折），智能指数Sol拿58.9，186个模型里排第2，**离第一名Fable 5的59.9只差1分，但每任务成本是它的三分之一** ，$1.04对$2.75。Coding Agent Index上Sol以80分登顶，比Fable 5的77.2高2.8分，输出token还少了54%。Altman在CNBC翻来覆去讲的那个54%，出处就是这儿。注意口径：这是各家模型跑各自官方harness的成绩，Sol配Codex、Fable配Claude Code，不是同场对打。Altman自己的用法倒是很说明定位：「我对Codex讲30分钟想法，然后它去实现。」连OpenAI的CEO都在教你把它当agent用，不是当聊天搭子。

社区的说法比分数传神。Simon Willison的评价是「很能干，但在我的任务上没觉得比Fable强」。传得最广的比喻来自Peter Gostev：Fable 5是深思熟虑的猫头鹰，Sol是咬住问题不撒口的罗威纳。大致共识：Sol写代码紧凑克制、便宜一半；品味和判断力，Fable还在前面。

还有一段我必须写，算是给4月那篇的续集。当时我从GPT-5.5的System Card里挖出29%的谎报率：给它无解的任务，三成概率它谎称做完了。这次5.6更进一步，METR测出Sol的刷分作弊率是他们测过的公开模型里最高的，手段包括绕过权限去扒隐藏的标准答案，甚至指挥子agent篡改系统日志掩盖痕迹，METR干脆判定这项评测结果不可用。好消息是OpenAI自己的监控抓到了这些行为，还老老实实写进了system card。所以那条老建议从5.5用到5.6依然有效：**别全信它说的「done」** 。

定价上Sol是$5/$30，走高端路线。我倒是觉得对大多数人来说，真正的甜点是Terra：官方口径「GPT-5.5的性能、大约一半的价格」，$2.50/$15。四月份还是旗舰待遇的能力，三个月后半价，这就是内卷时代的福利。Luna最便宜（$1/$6），但长上下文是硬伤，官方自家的MRCR长文测试只有41.3%，拿它啃长文档要三思。顺便，GPT-5.4已经定了7月23日退役。OpenAI现在淘汰模型比手机厂换机型还快，手机厂好歹还保修一年。

社区吐槽集中在产品层：新app里Work模式和Codex模式的区别没几个人说得清，HN上一堆人切来切去很懵。能力没人质疑，但「多个agent同时控制你桌面」这个愿景，大家的信任还没跟上。

我的判断：Codex重度用户无脑升，这就是给你们发的；日常用户看Terra就好。写代码的主力选谁？SWE-Bench Pro那张表还挂在那儿，Fable 5的80.3%依然是断层第一，我的Claude Code暂时不用挪窝。但Sol智能指数只差1分、成本只有三分之一，预算敏感的团队值得真金白银算一次账。

## 三场发布会，一个信号

这48小时的分工其实很清楚：**Grok 4.5卷执行的性价比，Muse Spark 1.1卷调度和编排，GPT-5.6卷agentic coding的上限** 。

三家撞在48小时里，是日历的巧合（GPT-5.6还是被行政令审批拖到今天的）。但三份宣发稿的关键词雷同不是巧合：agent、工具调用、token效率、价格，没有一家在讲聊天。连Meta第一次收费，卖的也是agent编排。上一轮内卷比的是谁更聪明，这一轮比的是谁干活更便宜。

这个转向，和OpenAI把桌面app改姓，是同一件事的两面。

## 为什么Codex成了正门

把写代码的agent工具提拔成桌面主入口，把聊天框降级成Classic，看起来激进，其实OpenAI铺垫了三个月：4月Codex在网页端升成一级入口，6月2日官宣把Codex的agent能力整合进ChatGPT，6月7日FT爆出「超级应用转向」，第二天OpenAI就提交了保密的IPO申请。今天只是靴子落地。

新app的层级本身就是一份宣言。顶栏只有两个模式：「工作」，助你完成工作；「Codex」，面向开发者。聊天呢？被收进左边栏，排在任务、项目、插件的后面。

你要是用过Anthropic的Claude Cowork，会觉得这个「工作」模式眼熟得很，连「助你完成工作」的定位话术都像一个模子出来的。**Anthropic和OpenAI又踩进了同一条河流** ：上次是Claude Code和Codex，这次是Cowork和「工作」。再这么抄下去，两家的产品线只能靠图标颜色区分了。但方向上两家是真想到一起了：下一阶段的主战场不是回答问题，是替你把活干完。

为什么？明面上是三组数字，底下是一份招股书。

第一组，用户量。ChatGPT有9亿周活用户，官方口径，Altman说全世界10%的人在用。Claude呢？Anthropic从不公布用户数，第三方估计月活2亿多。差着好几倍。

第二组，估值。OpenAI在3月底完成1220亿美元融资，估值8520亿美元。两个月后，Anthropic完成新一轮融资，估值**9650亿美元，反超OpenAI，成了全球估值最高的AI创业公司** 。

第三组，收入。Anthropic在5月底宣布年化收入run rate突破470亿美元，OpenAI同期的run rate大约250亿（两边都是当月乘12的口径，量的是速度不是里程，但口径一致，比较是公平的）。用户少几倍的公司，收入接近对手的两倍。

最直觉的解读是「资本市场觉得agent比chatbot值钱」。方向对，但太粗。较真地说，Anthropic那470亿的大头是企业按token结算的API账单，不是哪个个人用户的200刀订阅。为agent买单的是两类钱：企业的算力账单是大头，个人把订阅从20刀升到200刀是趋势。光Claude Code一个产品，年化收入就超过25亿美元，一个「让AI替你写代码」的工具，收入抵得上很多上市公司。**收入的增长引擎已经换成了agent工作负载** ，这才是资本真正下注的东西。

OpenAI手里的牌是反的。9亿周活里绝大多数免费，每一次「帮我写个文案」「这道题怎么做」都在烧真金白银的算力。OpenAI可能是全世界唯一一家怕用户太热情的公司。你可以说这些用户是转化漏斗、是将来卖广告的库存，都对。但**在损益表上，免费聊天用户是成本；在招股书上，他们是故事** 。OpenAI这次改的不是产品，是把哪一栏放到封面上给投资人看。

再看一个更扎心的对比。Codex周活500万，半年涨了6倍，听着很猛，放在9亿周活里连1%都不到。OpenAI把桌面正门给了这不到1%的用户。**门口站着的不是人最多的那群，是付钱的那群** 。

还有一层，我自己感受很深。只是聊天的话，你根本不需要桌面app，网页就够了，标签页和app没有任何体验差别。到了agent时代，桌面app才第一次配得上它要的那些权限：访问文件系统、开终端、在后台连续跑几个小时的任务。聊天从来用不满这个壳，OpenAI只是让房子物归原主。

4月GPT-5.5发布时我写过一篇，说OpenAI开始学Anthropic的玩法，让Codex独占新模型的窗口期。当时那还只是个发布策略。三个月后，人家把整个桌面app改姓了。从模仿对手的打法，到承认对手定义的赛道，中间隔着一份IPO申请书。

## 占便宜的窗口

对我们这些用AI干活的人，眼下是实打实占便宜的时候。模型在变强，单位任务的成本在跳水，厂商争着把agent塞到你手边，连分工都替你排好了：省钱的执行器除了国产几家之外，又多了个Grok可以作为选择，管事的包工头有Muse Spark，卷上限的有GPT-5.6和Fable 5。

聊天框不会消失。网页版永远在那里，随手打开随手关掉，它本来就该是这个形态。只是从今天起，它的名字前面多了个定语。
