# 腾讯研究院AI速递 20260507

生成式AI

一、OpenAI推出GPT‑5.5 Instant，幻觉降52.5%，字数减30%

1.GPT‑5.5 Instant正式取代5.3 Instant成为ChatGPT默认模型，对话语气更自然，可调取过往记忆补充背景信息；

2.内部评估显示，在医学、法律、金融等领域幻觉率下降52.5%，文档解析、博士级问答、AIME 2025等基准全面提升；

3.新模型回答更紧凑，字数减少30.2%，记忆来源功能全系上线，用户可追溯并管控个性化回答的上下文依据。
