Agentic AI 的技术快车，加速推进行业智能体或通用智能体平台的落地。 作为 AWS 核心级合作伙伴，小宿科技始终聚焦企业 AI 转型需求，凭借安全可靠、高效敏捷的技术架构，为企业提供 Agentic AI全生命周期所需的 “数据 – 模型 – AI 云” 全栈基础设施服务，助力客户打通从技术探索到业务落地的关键链路。然而，在服务数以百计各行业客户的实践中我们发现，当企业真正着手开发、测试、部署并运维一个生产级别的安全可靠 Agentic AI 项目时，仍需跨越重重挑战，具体包括： 不同场景或不同任务需要不同的大模型，对接大模型种类多，对接复杂； 大模型无法获取实时最新信息，目前仍存在一定程度的幻觉； Agent开发框架繁多，如何选择一个合适的框架； 不同应用场景，需要集成大量MCP工具，传统应用只支持REST API，非标准化MCP接口，Agent无法直接使用； Agent应用开发过程中，企业希望更多聚焦于AI应用的快速迭代，Agent应用开发完成后，如何大规模、安全可靠的生产部署，如何实现企业级隔离保障的安全运行环境，基础设施如何自动扩展等； AI技术日益月异，员工针对AI应用开发的技能储备不足，如何中快速上手AI Agent开发，补上技能短板 1.1 解决之道 针对Agentic智能体落地中的痛点，借助亚马逊云科技云上一站式GenAI解决方案及相关服务，包括Amazon Bedrock、Amazon Bedrock AgentCore、Strands Agents、Kiro及小宿智能搜索，通过提供实用的AI Agent 服务，从开发、构建、到大规模安全部署，端到端的赋能企业，帮助企业实现AI 项目的快速落地。 1.1.1 Amazon Bedrock 模型服务 Amazon Bedrock 集成了来自 AI21 Labs、Anthropic、Cohere、DeepSeek、Luma AI、Meta、Mistral AI、Stability AI、TwelveLabs、Writer 和 Amazon 等领先人工智能公司的100多款高性能模型，从而为Agent模型接入提供多种选择，并且无需重写代码即可轻松更换这些模型，使您能够随着需求的变化灵活地进行构建Agent 1.1.2 Amazon Bedrock AgentCore AgentCore包括：运行时、内存、网关、身份、可观测性和工具等，提供专为动态代理工作负载构建的基础设施、可增强代理功能的强大工具，以及适用于现实部署场景的基础控件。 通过Agentcore，用户可以安全、大规模地部署和运行功能强大的Agent。 并且兼容任何框架（包括 CrewAI、LangGraph、LlamaIndex 和 Strands Agents 等），并支持 Amazon Bedrock 内外的所有基础模型，能为您带来极大的灵活性。 AgentCore 可消除用户构建专用代理基础设施时千篇一律的繁重的维护工作，专注加快代理进入量产的过程。 1.1.2.1 AgentCore Runtime AgentCore Runtime 是一个安全的无服务器运行时，专为使用任何开源框架（包括 LangGraph、CrewAI 和Strands Agents）、任何协议和任何模型来部署和扩展动态 AI 代理和工具而构建。Runtime 专为代理工作负载而构建，具有业界领先的扩展运行时支持、快速冷启动、真正的会话隔离、内置身份以及对多模式负载的支持。开发人员可以专注于创新，而 Amazon Bedrock AgentCore Runtime 则负责处理基础设施和安全问题，从而加快产品上市时间。 1.1.2.2 AgentCore Memory AgentCore Memory 让您的 AI 代理能够通过保存即时和长期知识，提供智能、情境感知和个性化的交互。AgentCore Memory 提供两种类型的内存： 短期记忆 ：存储对话以追踪即时情境。 例如，假设您的编码代理正在帮助您进行调试。在会话期间，您要求它检查变量名称、纠正语法错误并查找未使用的导入语句。代理会将交互作为短期事件存储在 AgentCore Memory 中。之后，代理可以检索这些事件，以便无需您重复之前的信息即可进行对话。 短期记忆捕获原始交互事件，维护即时情境，支持实时对话，丰富长期内存系统，并支持构建高级情境解决方案，例如多步骤任务完成、会话中知识积累和情境感知决策。 长期记忆 ：存储提取的洞察信息（例如用户偏好、语义事实和内容摘要），以便在会话之间保留知识。 用户偏好——想象一下，使用 AgentCore Memory 的编程代理是您长期的编程伙伴。经过许多天的实践，它会注意到您始终编写简洁的代码并带有注释，喜欢使用蛇形命名法 (snake_case)，使用 Pandas 进行数据分析，并在最终确定函数之前进行测试。下次，即使经过多次会话后，当您要求它编写数据分析函数时，它也会自动遵循存储在 AgentCore Memory 中的这些偏好，而无需您再次告知。 语义事实——编程代理还会记住“Pandas 是一个用于数据分析和处理表格的 Python 库”。当您询问“哪个库最适合处理表格数据？”时，它会立即推荐 Pandas，因为它从语义记忆中理解了 Pandas 的含义。 内容摘要——编程代理会生成会话摘要，例如“在本次交互中，您创建了一个数据清理函数，修复了两个语法错误，并测试了您的线性回归模型”。这些摘要既能跟踪已完成的工作，又能压缩对话上下文，从而能够高效地引用过去的活动，并优化上下文窗口的使用率。 1.1.2.3 AgentCore Gateway 借助AgentCore Gateway，几行代码，即可将现有API快速转变为支持代理的MCP工具，简化agent与各类工具的连接流程，提升agent执行各种任务的能力。 Amazon Bedrock AgentCore Gateway 是 亚马逊云科技推出的全托管工具管理中枢，专为 AI 代理设计。它能将 API、Lambda 函数、第三方服务（如 Salesforce、Slack）快速转化为符合 Model Context Protocol (MCP) 的工具，同时为 AI 代理提供统一、安全的工具访问入口，解决 “工具集成难、管理杂、安全风险高” 的核心痛点。 AgentCore Gateway主要功能： 多类型工具接入：支持 OpenAPI 规范、Lambda 函数、Smithy 模型导入，及热门工具一键集成，无需重构现有资源； 统一访问与智能发现：单端点管理所有工具，内置语义搜索，帮代理按任务上下文匹配工具，减少调用复杂度； 全方位安全认证：覆盖入站（代理身份校验）、出站（工具凭证管理）双向 OAuth 认证，自动刷新令牌、安全存储凭证； 协议翻译与框架兼容：自动转化 MCP 与工具协议，兼容 CrewAI、LangGraph 等开源框架及任意 AI 模型； 全托管运维：Serverless 架构弹性扩缩容，提供调用日志、监控指标，零基础设施管理成本。 核心价值 降本提效：几行代码完成工具转化，替代数月级自定义开发，缩短 AI 代理落地周期； 安全可靠：企业级双向认证与合规保障，避免凭证泄露和未授权访问； 灵活扩展：支持工具规模化管理（从数十到数千个），兼容多框架 / 模型，适配业务增长； 零运维负担：亚马逊云科技 负责底层维护与升级，开发者聚焦代理核心能力创新。 1.1.2.4 AgentCore Identity Amazon Bedrock AgentCore Identity 是一项专为 AI 代理和自动化工作负载设计的身份和凭证管理服务。它提供安全的身份验证、授权和凭证管理功能，使代理和工具能够代表用户访问 亚马逊云科技 资源和第三方服务，同时帮助维护严格的安全控制和审计跟踪。代理身份作为具有专用属性的工作负载身份实现，这些属性支持代理特定的功能，同时帮助保持与行业标准工作负载身份模式的兼容性。该服务与 Amazon Bedrock AgentCore 原生集成，为代理应用程序（包括主机代理或使用 Amazon Bedrock AgentCore Runtime 和 Amazon Bedrock AgentCore Gateway 的工具）提供身份和凭证管理； 1.1.3 Strands Agents Strands Agents SDK 采用模型驱动（Model-Driven）的设计理念，充分发挥先进大模型在规划、思维链（Chain-of-Thought）、工具调用及自我反思等方面的能力，显著地简化了智能体的开发流程。开发者仅需定义核心提示词（Prompt）和可调用的工具列表（Tools），即可快速构建功能智能体（Agent），并能将本地测试成果无缝部署至云端环境。 在模型生态方面，Strands Agents SDK 小宿智能搜索是专为Agent设计的智能搜索与数据服务，提供多语言、多模态、多能力的数据获取与处理服务。针对Agent对信息精度、实时性和多模态处理的需求，小宿科技专门为 Agent 设计了智能搜索与数据服务，原生支持35+主流语种；网页长摘要总结，以 Markdown 等格式输出，针对Agent 友好，Agent 一次调用中获取全量信息，降低大模型幻觉；多模态搜索，一站式满足多样数据需求；智能召回 + 精准排序，内容质量远超主流搜索；提供秒级响应，高可用。 核心功能 基础通用搜索：自研的基于语义的网络搜索引擎，能挖掘常规搜索引擎无法解锁的数据维度，为AI 提供了更精准的知识获取通道； 全文展示搜索：针对传统搜索引擎仅返回简短摘要的问题，全文展示搜索检全网信息可直接呈现完整内容，并自动高亮关键信息，用户可在结果页一站式获取核心内容； Reader读取产品：支持各类公开合规页面的抓取读取。使用便捷，输入目标url即可返回全文信息，对静态页面、动态页面均能完成读取，满足多种输出格式，性能优越； 产品优势 高稳定性：专为构建高度稳定、可靠和可扩展的应用而设计，在每秒处理 100 多次查询 (QPS) 的同时，延迟低于一秒，并保证 99.9% 的服务可靠性 易用性：只需两行代码即可快速集成、兼容必应标准 API 和多个搜索引擎 高效率：小宿自研的搜索解决方案只需一次点击即可提供完整的网页内容、通过 AI 突出显示关键信息，消除额外页面访问或二次数据采集的麻烦 全场景应用：将小宿的搜索功能与平台的大语言模型服务相结合，可快速构建各种应用程序，包括 AI 深度研究和基于 RAG 的客户服务等。 全栈部署：通过灵活选择小宿提供的多种大语言模型与搜索产品，支持面向不同场景的一站式快速部署。 1.2 架构设计 1.2.1 设计原则
