# 曾经爆火的小龙虾凉了？现在圈内都在用这款新 Agent

> 数据派THU 2026-06-09 17:00 报道，转发新智元原稿。同一清华 PilotDeck 开源事件的第 2 中文译本，与 2026-05-28 ASI启示录译本形成多公众号传播。

## 三大核心场景实测（数据派 THU 视角独到之处）

### 1. 跨域 WorkSpace 并行：奶茶店 + 数据大屏 + 性格测试

同时开两个 WorkSpace 并行跑：

- **奶茶店模拟经营游戏**：核心循环 5 款奶茶 / 进货 / 定价 / 排队 / 财务五子系统，清新卡片风 UI，自写 JS 模块
- **AI 融资数据大屏**：4 个图（融资总额 TOP 10 / 北美欧洲亚洲占比 / 通用AI/企业AI/生成式AI赛道分布），动画 + 悬停详情
- **程序员性格测试**：10 道题贴近真实开发场景，6 种人格（架构师🏛️ / 搬砖侠🧱 / 完美主义者✨ / 魔法师🧙 / 布道师📣 / 哲学家🤔），GitHub 暗色 + JetBrains Mono 视觉

**核心论据**: 三个项目跨度大到离谱，但在 PilotDeck 里都是独立 WorkSpace，各跑各的。

### 2. 三层舱结构：文件系统 + 记忆 + 技能

- **专属文件系统**：哪些文件归这个项目、AI 生成了什么，边界清楚
- **专属记忆**：Project Memory（项目定义+进度）+ Collaboration Feedback（用户偏好），全可改可追源
- **专属技能**：Skill 应用商店一键安装到对应 WorkSpace（game-asset-finder → 游戏舱，minimax-pdf → 文档舱）

**对比叙事**: "别家的 WorkSpace 是文件夹加静态规则。PilotDeck 的 WorkSpace 是 AI 的完整生存环境。"

### 3. Token 账单砍一半（具体数据 + 路由机制）

| 场景 | 不开路由 | 开路由 | 节省 |
|------|---------|--------|------|
| 程序员性格测试 | $10.97 | $1.42 | **75%** (省 $9.55) |
| 社交媒体（小红书内容） | $12.58 | $2.83 | **70%** |
| 复杂任务（播客多语言+金融分析+代码文档）| $18.36 (Sonnet 4.6 单体) | $3.15 (Sonnet 4.6 主 + MiniMax-M2.7 子) | **83%** (1/6 价格，效果略好) |

**核心机制创新**: 子 Agent 级别路由（非 request 级）→ 保持 KV-cache 连续性，避免「读档」开销。规则支持自然语言描述：「代码相关的子任务走 Claude Opus，文本处理走便宜模型」。

### 4. 端侧模型自动调度（独到技术细节）

PilotDeck 可以接本地部署的模型做子 Agent，敏感数据不出本机。**关键创新**: 任务需要时会**自动部署端侧模型**干活，例如播客多语言处理时自动装 VoxCPM 生成语音。也可以云端模型思考 + 本地模型执行，省钱+隐私双解。

### 5. 记忆白盒 + Dream + Rollback

Memory 面板每条记忆标时间戳/来源路径/类型。记错点进改，冲突直接删，不要重启对话。Dream 机制（AI 空闲后台整理记忆）失败时可一键 Rollback Last Dream 回到整理前状态。

## 立场与叙事

把 OpenClaw 比作「飓风，第一次把 Agent 范式吹进大众视野」，但**没成为 Linux**——跑得太快没建足够代码壁垒和生态就被拍在沙滩。把 PilotDeck 比作「清华系高材生」，可以「把隔壁小龙虾拍在沙滩」的「真正面向纯粹生产力的智能体协作舱」。

**关键论断**: 「大模型浪潮从 ChatGPT 到 OpenClaw 极客玩具，Agent 形态一直在快速迭代。**但直到今天，当『成本』『隔离』『记忆』这些纯工程化、偏后端的硬核痛点被 PilotDeck 逐一击破时，Agent 才算真正落地了。**」
