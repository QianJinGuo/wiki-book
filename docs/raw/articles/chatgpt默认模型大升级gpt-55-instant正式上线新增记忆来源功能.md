# ChatGPT默认模型大升级，GPT-5.5 Instant正式上线：新增记忆来源功能
↑阅读之前记得关注+星标⭐️，😄，每天才能第一时间接收到更新
OpenAI宣布，ChatGPT的默认模型正式升级为GPT-5.5 Instant，面向所有用户开放。说人话的GPT终于又回来了
###  更准确，废话更少
准确性是这次升级的核心。
在内科、法律、金融等高风险领域，GPT-5.5 Instant的幻觉内容比GPT-5.3 Instant减少了52.5%。针对用户此前专门标记过存在事实错误的对话，错误率下降了37.3%。
能力也全面提升，包括图片和照片的分析、STEM问题的解答，以及判断何时需要联网搜索以给出更有用的回答。
回答变得更短，但信息量不打折。GPT-5.5 Instant的回复用词减少了26.6%，行数减少了36.4%。它会减少不必要的追问，也不再滥用emoji和过度格式化。
###  更懂你：记忆来源
个性化能力同样有所提升。
GPT-5.5 Instant能更有效地调用历史对话、上传的文件，以及已连接的Gmail内容，在需要个性化的时候主动使用这些上下文，减少用户重复说明的频率。
与此同时，OpenAI在所有ChatGPT模型中引入了记忆来源功能。当回答经过个性化处理时，用户可以看到具体调用了哪些上下文，比如已保存的记忆或某段历史对话，并可以直接删除或修正过时的内容。
如果分享对话，记忆来源不会对外显示。用户也可以通过删除历史对话、在设置中修改已保存记忆，或使用临时对话模式来管理自己的隐私。
OpenAI说明，记忆来源展示的是最相关的部分，不一定覆盖所有被参考的内容，后续会持续完善。
###  上线节奏
GPT-5.5 Instant从今天起向所有ChatGPT用户推送，替换GPT-5.3 Instant成为默认模型，同时在API中以chat-latest提供。付费用户可以在模型设置中继续使用GPT-5.3 Instant，保留期限为三个月，之后正式退役。
基于历史对话、文件和Gmail的增强个性化功能，目前正在向Plus和Pro用户的网页端推送，移动端即将跟进，后续计划扩展至Free、Go、Business和Enterprise用户。记忆来源功能正在向所有消费者计划用户的网页端推出，移动端也即将上线。
