# Fable 5 实战之：五亿 token 暴改我三年前的网站

大家好，我是 John。

[昨天的文章](<https://mp.weixin.qq.com/s?__biz=MzA4NzgzMjA4MQ==&mid=2453486370&idx=1&sn=a14c3dd3cccc067fa915131eaa3752ac&scene=21#wechat_redirect>)介绍了 Anthropic 工程师 Thariq 的 Fable 5 实战指南，核心观点是：**Fable 5 的能力上限，取决于你能发现多少自己还不知道的东西。**

因为上周的工作日实在是有些太忙，而本周末也是 Fable 脱离订阅套餐前的（可能）最后一个周末，所以我也是刻意把 Fable 5 的额度留到了周末，决定要好好干上几件事来压榨一下它。

这其中的一件事，就是把一个我三年前开发的、基本上每天都会用的网站的前端来开刀大改版。（当时觉得搞的还行啊，但怎么现在越来越觉得难看了……  


01

## 三年前的 AGI Hunt

这个网站叫 AGI Hunt（没错，我公众号的同名网站），网址是：

https://i.agihunt.info/feeds/ai/2026-07-05/zh （为什么是这个奇怪的链接不是纯域名，下面会讲，因为有个 bug……）

它是我在 ChatGPT 刚出来那会儿做的了，到现在其实已经三年多了（时间过得真快）。

旧版首页

其实现在都还有些 bug，比如你直接访问 https://i.agihunt.info/ 是会报错的，但我也是真的懒，一直没（空）去修……

而当年做这个的原因也非常之简单：

ChatGPT 出来之后，每天的消息都快爆炸了，到处都是标题党、震惊体……，而且信息很乱，完全没有个好的顺序、章法和体系。同时信息源也太杂了，X、Reddit、HF、各种博客和论文……根本看不过来。。。

所以我觉得我需要一个每天能实时追踪 AI 领域的各种动态，能适合我获取信息、学习知识的内容结构和呈现逻辑，能分论文、算法、模型发布、行业……甚至，八卦……

于是我便做了这么个工具（其实还有个星球，不过不建议加了），用来实时采集 AI 相关的各类资讯，然后对同类内容进行相似度识别和聚簇展示。

聚簇详情

当然，后来我还给自己加了一堆机器人，用 IM 方式把消息推给我（比如时效性或其他一些垂类信息），但这个站点我还是经常会来点开看的。

我甚至也看到有人采集或参考我这小破站点做了自己的日报，做了自己的站点，我也毫不介意，有帮助就好。

而这个在 2023 年初做了之后，我基本就没再动过了，也算稳定运行。回想当时，还真的是太苦逼了，因为模型能力太差了，大约有 50% 的代码是我自己手搓出来的：

其中后端相关的逻辑其实没花我多少时间，算是经验自带的信手拈来。反而大头工作花在了前端页面的打磨上，比如：怎么把某个 div 调成居中、怎么在切换 cluster 卡片时有个凑合的动效……

相比现在，我手搓代码的比例已经是 0% 了。

02

## 未知的未知

于是这个有 Fable 陪伴的周末，我决定把这上面我极其讨厌的卡片式设计给做一下视觉改版（审美真的会变）。

结合 Thariq 说的方法论，我想就也来试一下自己的「未知的未知」。

我知道自己不喜欢现在的卡片式设计，但我不知道什么样的设计才是我想要的，甚至不知道有哪些可能性是我压根没考虑过的。

所以我刻意不去讲自己懂什么，不去引导，而是让 Fable 5 自己先发散。

我直接跟 Fable 说：

我想给整个前端完全重新做个改版，但我没什么视觉审美，也不知道有哪些更好的完全不一样的我不知道的可能性。请给我 4 个完全不同的设计方向，先做一个 HTML 文件，可以从服务器拉真实数据来模拟或 mock，让我先感受一下布局，再评估是否进行真正的应用。

四方向设计稿

Fable 5 先是自己去看了项目蓝图和现有前端，搞清楚各模块的定义和现有功能，再从服务器拉真实数据做 mock，最后并行启动 4 个 agent，各做了一个完全不同方向的设计稿。

其实，第一种我就挺满意了：

编辑部风格

有点报社报纸的那种风格，第一眼我就觉得很有些惊艳，非常的不 AI 也没用蓝紫色，甚至让我对后面有了更多期待。

但我想……要不再来几种？

十方向设计

于是让它又给了我十种。

所有方向总览

一共十几种方向，每一种 Fable 都做成了可以直接打开的 HTML 文件，用真实数据 mock 的，每个都能直接在浏览器里看效果。

## 三种心动

不过看完，我能看得上的大约也就三种，分别是 1、2、9。

**方向 1：AI 日报编辑部。**

编辑部风格

浅色纸感、衬线报头、多栏报纸网格，像是一份每天出版的 AI 情报纸。

我对这种风格是真的很喜欢（有点相见恨晚一见钟情的感觉）。

**方向 2：情报终端。**

终端风格

深色等宽、多面板同屏、trends 跑马灯，Bloomberg 式的信息密度。

其实还行，只是太极客了，不太适合日常阅读，酷是酷但有点费眼睛……而且信息密度太大了，阅读体验其实并不好。

**方向 9：新粗野主义。**

粗野主义

黑白粗框、超粗字重、荧光黄点缀，自信、直接、反精致。

虽然也还可以，但稍微有点花哨……我犹豫了下，最终还是放弃了。

而其他的几种就太普通了，基本上在我看来就是 AI 的平均值，没有什么新意，一眼便能知道是 AI 做的（平均审美）。

04

## 改版上线

最后，我还是决定用方向 1 的报纸风格。

因为这种设计完全去掉了那种已经被用烂了的、我其实有点讨厌的卡片式布局，换成了更适合阅读的报纸排版，看起来会让我觉得很舒服。

好了，让 Fable 动手！

我也忘了多久之后……

总之，现在，新版网站已经上线了：

新版首页

同时，我把之前没有在原网站上线、但在我的 Telegram Bot 里已经做好的渠道监测也加了进来：

•  Hugging Face 

•  GitHub 

•  Reddit 

•  YouTube 

跨平台聚簇与趋势

并且，我还对跨平台的信息做了内容的聚簇，这样不同平台的内容在讲同一件事情的时候，它们也能聚合在一起。

还搞了全网跨平台的热度公式权重对齐，从而可以把不同平台的点赞、评论、转发等数据按统一的公式做归一化，使得不同来源的内容可以放在同一个热度排行里做比较。

当然，上面的任务，我基本都只是说了下目标，并稍微暗示了下思路，用了几次 /Goal 大汗，Fable 就完成了。

感受就是，虽然 A\ 是真的 \，但 **Fable 5 确实还是太强了。**

05

## 不放鸽子了

从 23 年到现在，我估计我微信可能有上百人次问过我：能不能接入 AGI Hunt 的数据，我也是满口答应过很多次，或者就回复说：要不你找个 AI 爬一下我的页面……

总之，答应很爽快，但全都放了鸽子。

欠下的债，就让 Fable 替我来还吧……

其实也就几句话的事，Fable 就给我搞定了。接入页面、密钥管理、Agent 一键安装，全都有了。

接入页面

方式上非常简单，把下面这句话发给你的 Claude Code、Codex 或者任何能联网执行命令的 Agent 就可以了：

> “ 请帮我接入 AGI HUNT 实时 AI 资讯：阅读 https://agihunt.info/agent/v1/skill.md 并按其中的说明完成安装和授权。

然后就会发生下面的事：

1.  Agent 读取接入说明书，把它安装成自己的 skill 

2.  Agent 给你一个授权链接，打开它，用 Google 登录，点一下「授权」 

3.  Agent 自动获得专属密钥并妥善保存。之后直接问它「AI 日报」「AGI Hunt 最新模型动态」就好 

每个 Agent 设备一把密钥，互不影响，不再用的你可以随时吊销。

想让你的 Claude Code 或 Codex 接入 AI 实时快讯，把上面的句话扔给你的 Agent 就可以了。

但注意：我做了很多流控，请友善对待，不要暴力驾驶。

因为我还有个 AI 在 /loop 监测日志和流量，如果谁用猛了，它可能就上暴力手段了！

## Fable 的最后两天

这次改版如果放在三年前，我真的可能要干一个月。

但现在有了 Fable 5，我只是在做其他事的过程中，顺带着用语音跟它说几句话，而已……

就全搞定了，成本上，全程我看了下大约花了有小几百美金，大约 5 亿 token。

过程中我也尝试让 Codex 去 review 了几次 Fable 5 的代码，然后发现其实不太有这个必要……可能这坨代码的逻辑也不算那么复杂吧。

但遗憾的是，我们是时候得做好没有 Fable 5 的准备了：7 月 7 号（周二早上）起，Fable 5 就不能在订阅套餐里使用了，要用得自己掏钱了。

要真的自己掏银子用的话，确实还是太特么贵了，肉疼……

目前看来，只能指望两件事了：

1.  A\ 能把 Fable 5 的用量保留到订阅套餐里，现在的一半也行。。 

2.  GPT 5.6 赶快放出来吧！ 

另外，这个网站主要面向我自己使用，只是顺手分享出来，所以代码暂时没计划开源（没整理好……），如果有建议和反馈可以随时（让你的 AI）到 GitHub 上提 issue：
