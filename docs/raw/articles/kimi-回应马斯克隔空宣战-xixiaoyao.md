# Kimi 回应马斯克隔空宣战

上周五 Kimi K3 发布，72小时内引发了四场连锁反应。

第一波是模型实力本身，K3 的能力被普遍认为已经摸到了Fable 5水平线，前端编程中拿下 Frontend Code Arena 榜一，超过Claude Fable 5 和 GPT-5.6 Sol。

第二波是模型层的价格重估，K3的定价虽不是 DeepSeek 式的白菜价，但是已经夯实了“前沿平替价”的用户心智。

第三波，叙事升级。开始有一拨人讲中国AI已经追平了美国顶尖AI模型， 中国的开源模型已经影响到了美国AI公司的估值。

第四波，算力层直接地震了。

7月19日，有用户发现在Kimi的订阅页面中，四个档位的会员齐刷刷显示已售罄。

不是产品出了问题，也不是被封禁，官方给的理由是——

因为K3需求远超预期，已经逼近现有集群承载极限，因此暂停 C 端新用户订阅，把现有算力优先保障给已订阅用户。

一个AI会员硬生生卖断货了。推特上同步发出的帖子，不到半天，浏览量迅速突破 1000 万。

不过，在官宣暂停订阅前，月之暗面还短暂更新了一版 Kimi 会员订阅价格表。

先是原本199/月的第三档，飙升至699，大幅涨价。

八点左右，官方又撤回了一个涨价，页面恢复原样。

先涨价，又停售、然后又撤销涨价，截至目前官方还没给个解释。

时间回到7月16日，月之暗面发布 Kimi K3，2.8 万亿参数，全球最大的开放权重模型，100 万 token 上下文，原生多模态。官方跑分直接对标 Opus 4.8 和 GPT-5.6 Sol，第三方盲测平台 Code Arena 上，编程能力干到全球第一。

在推特上，马斯克直呼“impressive”（令人印象深刻）。

3月份的时候，马斯克点赞了Kimi的底层架构，这是第二次公开认可Kimi，还将 Kimi 视为了竞争目标，宣布他们公司正在训练2万亿参数新模型，将于下周完成初步训练，并称其全面优于1.5万亿版本，且“可能超越Kimi”。

面对马斯克的隔空宣战，Kimi 官方回应——“欢迎加入两万亿俱乐部”。

2万亿规模是当前大模型参数竞赛的“前沿门槛”，单次完整训练的算力成本可能达到数亿美元级别，全球有能力、也有意愿把总参数推到这个量级的玩家屈指可数。

目前公开信息中达到或逼近这一规模的，主要有下面几个模型：

  * **Kimi K3（月之暗面）** 2.8万亿总参数，目前全球参数最大的开源模型。
  * **Qwen3.8-Max（阿里通义）** 7.19阿里在上海 WAIC 上预告了这款模型，2.4万亿参数，离K3发布仅过去72小时。
  * **MiniMax M3 Pro** 参数量据业内估计约 2.7 万亿（外界预估，官方未确认口径）。
  * **Grok 4.6（xAI）** 2万亿参数，马斯克 7 月 18 日确认本周完成初步训练，尚未发布。
  * **Claude Fable 5、GPT-5.6 Sol 等闭源旗舰** 参数规模从未披露，外界普遍推测在这一量级或以上。

除了参数量外，推特上有网友还放出了几家模型的实力预估——

发布这几天，推特上也已经攒了一堆K3的实测。

推特网友 Romain Torres 用 Kimi K3 和 GPT 5.6 sol 分别生成了武士抽刀的画面，点开一看，我感觉无论是流畅度还是画面质感，K3可以说和GPT 5.6 sol 不相上下。评论区的高赞评论夸的是Kimi K3： 生成的画面纹理比GPT 5.6 sol 更加均匀。

视频详情 

从效果来看，说Kimi K3对标 Opus 4.8 和 GPT-5.6 Sol，还真不是吹。

价格方面，Kimi K3 延续了国产模型一贯的优点——高性价比。在能达到同等效果的情况下，价格是这样的：

  * **Kimi K3：** 输入 3 美元/百万 token，输出 15 美元/百万 token；
  * **GPT-5.6 Sol：** 输入 5 美元/百万 token，输出 30 美元/百万 token；
  * **Opus 4.8：** 输入 5 美元/百万 token，输出 25 美元/百万 token。

虽然输入价差距不算大，但输出价差异悬殊，而 Agent 干活恰恰是输出大户，等于 K3 在最烧钱的那一项上，给打了五折。

能力又强，价格还便宜，用户大量涌入，请求量逼近集群承载极限也就不奇怪了。

**爆火的 K3，实际干活怎么样？**

目前，我手头的199老套餐账号还能正常续费、正常支付。K3发布后，我也拿用它实打实的干了几天活。

  1. 最惊艳的是前端和视觉。k3一把梭生成的页面完成度非常高，审美在线。完全配得上Code Arena的全球第一。
  2. 短板是有点慢。同一个任务，别家五十分钟，kimi code得做上小一个半小时，结果对，但过程有点熬人。
  3. 判断力、规划产品这种要「拿主意」的活儿，k3尚且不如Fable 5老练——不过，明显强过GPT-5.6。

很像一个刚进公司的天才工程师。

页面做得漂亮，技术活完成度很高。不过在让它决定产品应该往哪里走、什么功能该砍掉，它又会显得谨慎，甚至有点犹豫。总的来说，体感和榜单基本对得上。

与此同时，就在售罄的同一个周末，彭博报道月之暗面已向股东发出决议，寻求赴港上市，最快 6 个月内完成 IPO，新一轮融资估值或超 300 亿美元。

彭博引述知情人士称，月之暗面的年度经常性收入（ARR），已经从4月的2亿美元增长到6月的3亿美元，两个月增长50%。粗略计算，估值已经接近ARR的100倍。

但与数亿美元形成鲜明对比的，是社交媒体上流传的一张照片：Kimi K3 发布前两天的深夜，月之暗面的团队成员仍在办公室里埋头工作。照片中的办公环境并不奢华，甚至有些朴素。然而，正是在这样一间普通的办公室里，诞生了让全球重新审视中国大模型竞争力的 Kimi K3。

对于月之暗面而言，K3之后，下一段崭新旅程，才刚刚开启。
