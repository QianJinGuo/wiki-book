# Claude Code 太折腾？这个国产开源 AI 编程平台打开就能用

最近冲浪发现了一个Agent产品，叫MonkeyCode。

如果你想做一个网站，小工具，或者demo。但又不想安装CodeX, Claude Code这些工具，学习这些工具怎么使用。

那么MonkeyCode可能比较适合你。

因为MonkeyCode免费使用，无需安装，内置云端开发环境，并支持业内最全的顶尖大模型。

##  MonkeyCode是什么？

MonkeyCode 是一款开源的企业级 AI 开发平台，内置了开发环境管理、AI 模型管理、AI 任务管理、项目需求管理等能力，区别于其他的 vibe coding 工具，

MonkeyCode 是真正面向专业开发团队的 AI 助手。

这个是首页面，可以邀请好友得积分，一次5000积分，一次任务差不多消耗200积分左右，还是够用的。

可以免费使用国内主流基础大模型。每天3千万免费额度，日常写一些任务还是可以满足的。

旗舰模型也是可以使用的。

这里可以绑定你的代码仓库地址。

支持多平台仓库，我这里选用国内的Gitee，当然主流的还是Github。

完全开源：核心代码全部公开在 GitHub。

> https://github.com/chaitin/MonkeyCode/

现在已经有4.3k的星星。

感兴趣的朋友也可以去点个赞，支持一下国产Agent。

##  工具制作

最后测试做一个HTML转PNG图片的工具。这个工具我觉得还有一定的使用范围。

HTML 本质上是“可编辑的页面”，适合做复杂排版、卡片布局、时间线、表格、信息图、课程页。 但很多平台并不能直接展示 HTML，比如公众号正文、小红书、朋友圈、PPT、海报素材区，它们更接受图片。所以需要把 HTML 最终导出成 PNG。

利用下面这段提示词进行制作。

> 请帮我制作一个完整可运行的网页工具，使用 HTML、CSS、JavaScript 单文件实现。 工具名称： HTML 转 PNG 图片工具 制作一个浏览器端小工具，用户可以输入或粘贴 HTML 代码，然后将渲染后的页面内容导出为 PNG 图片。

完整版提示词

同样提示词有点多，因为我把页面布局的提示词也给到MonkeyCode了。所以会多一些。

几分钟后，工具网站就搭建好了。

左边就是HTML代码存放地方，右边是预览图片。还可以设置清晰度最高可以达到4K效果。

导出的效果。

制作一个模型科普的HTML页面，包含公司模型，市值，成立时间这些基础信息。然后把这个HTML转为PNG图片方便传播。

效果看着还行吧，有什么其他的小工具制作，可以直接在MonkeyCode里面进行制作。

##  最后

MonkeyCode解决了两个最直接的问题，不用下载安装环境，不用部署环境。

这对很多想快速试想法的人来说，门槛会低不少。比如做一个网页小工具、一个数据展示页面、一个小游戏 Demo，直接在云端写、云端跑、云端看效果，整个流程会轻很多。

当然，它也不是说能一次性把所有东西都做到完美。像游戏美术、页面细节、交互体验这些，还是需要不断提需求、反复优化。但从这几次测试来看，MonkeyCode的完成度已经够日常折腾一些小项目了。

如果你平时也想做点网页、工具、Demo，又懒得折腾本地环境，可以试试看。
