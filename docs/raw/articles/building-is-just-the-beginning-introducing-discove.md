# Building is just the beginning: Introducing Discoverability
![Image 1: Building is just the beginning: Introducing Discoverability](https://lovable.dev/cdn-cgi/image/width=3840,f=auto,fit=scale-down/content/news/covers/seo-launch-v2.png)
Author: Cecilia Stallsmith, Chief Marketing Officer at Lovable
I am a firm believer that there has never been a better time in history to solve problems. And yet, getting your solution in front of the right audience has (potentially) never been harder.
Today we’re introducing discoverability capabilities directly within the Lovable product experience. Your app now ships with the ability to be read and ranked by search engines, and with [Semrush](https://www.semrush.com/) built in, you can see how it's showing up across search.
Search is no longer dominated by a single channel. It’s fragmented, AI-mediated, and evolving faster than our playbooks can keep pace with.
Discoverability and brand visibility used to be a marketing problem to hand off _later_. But in a world where there are very few moats left to compete on, there is no _later_.
Here’s what this new functionality looks like in practice:
### Built to be discovered from day one
Every new app built on Lovable now is fully readable to search engines from the moment you publish. To be precise, they ship as complete HTML that search engines and LLM crawlers can read and index immediately.
And if your project is a part of the 40 million apps already built, you're covered too. Pre-rendering is now live across all existing apps, generating static HTML snapshots of your pages so crawlers can access your content without any action required on your end.
All apps are also structured so AI tools can read them as easily as humans can. Because being discovered on ChatGPT, Claude, or Perplexity is just as important as ranking on Google.
### World class search intelligence, built in
Lovable has partnered with [Semrush](https://www.semrush.com/), the industry's leading search intelligence platform, natively inside the building experience. Ask Lovable how your app is performing in search, find out which keywords are driving traffic, identify gaps in your content, and spin up optimized landing pages, all backed by Semrush's dataset of 28 billion keywords and 43 trillion backlinks. All in the same chat you build in. No separate account or tools required.
### See what's working, fix what's not
Lovable can now run an SEO review before or after publishing: checking sitemap, robots.txt, metadata, content structure, alt text, canonical tags, and more. A new SEO Dashboard replaces the Speed Dashboard, bringing your search insights and best practices into one view. Spot an issue, fix it in one click.
Over 200,000 projects ship on Lovable every single day. Now every one of them is built to be discovered. Go build!
