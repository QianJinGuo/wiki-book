# Better Call Sol: The Workhorse — OpenAI GPT-5.6 Sol Deep Analysis

OpenAI's GPT-5.6 Sol is finally here, along with the cheaper Terra and Luna.

## Sol vs Fable: The Workhorse vs The Architect

Zvi's core framing: Fable is the smarter one — the collaborator, architect, planner, and manager. Sol is the workhorse — the go getter, the place where if it's known what to do then it just gets done. Both are excellent models with very different profiles.

In terms of raw intelligence and "big model smell," Fable still has a substantial edge. It also seems to be better aligned and more trustworthy as an agent with less tail risk. Zvi still considers Fable "the best" model.

Sol has the edge in getting many practical things done, including computer use and web search. The recommendation: send identical queries to both and compare.

## Benchmark Positioning

Sol leads on Agents' Last Exam, AA Agent Coding Index v1.1, and BrowseComp — benchmarks that favor task-path clarity, step-splittable scenarios (CLI, terminal testing, browser automation). Fable leads on SWE-Bench Pro (open-ended repository-level code modification), where Sol trails significantly (64.6% vs 80%).

## Practical Usage Patterns

- Sol + Codex vs Fable + Claude Code — different tool combinations for different workflows
- Sol excels at well-defined, executable tasks
- Fable excels at architecture, planning, and open-ended creative coding
- Both should be used in concert for best results
