#  如何谈合作、找工作
我公众号的后台经常会收到各类消息，也会有直接加我微信好友的朋友，会给我提出各类诉求或合作。比如能不能投稿自己的文章，能不能介绍一下自己的项目，或者宣传一下自己的产品，或者问我公司还缺不缺人，说自己非常需要一个学习的机会。
几乎每天都会有。这里也抱歉我消息太多，有些朋友发来的消息我还没来得及回……
其实我自己，也非常能感受和体会这些朋友想要推进自己喜欢的事的激情，因为我也是过来人，不对，我现在也一样，也处于这样的状态之中。
但我觉得，这儿有一个点，是很多人可能没有意识到的：
当你提出一个诉求的时候，你往往在想的是，是否能得到一个对自己有利的结果，而容易忽视了对方可能的付出或负担，或者对方可能得到的收益。
用同理心来换位思考一下：当你对一个人提出某个诉求时，他会被什么打动来给你回应呢？
只要你想清楚了这个问题，那么有些话可能就不用说了，或者换另外一种方式去说，那可能就，很轻松就打动对方了。
这既为对方（比如我）节省了时间，也为自己节省时间，大家都有更多时间省出来了。
从某个角度来说，合作，或者说好的合作，应该是双方都能“占到便宜”的。它不应该是一个零和游戏，不是你输我赢或我赢你输，也最好不是“聊了个寂寞”，碰完之后双方都觉得浪费了时间。
再次推荐《高效能人士的七个习惯》这本书，其中的第四个习惯是：双赢思维。
当然，有很多好的项目，我也很乐于去学习，并愿意分享给更多人，让他们看到后有所收获。
但如果你的项目没那么好，或者我没那么看懂或认同，那我显然是不太愿意的。我自己也有过很多不够成熟的、甚至是 SB 的想法，我也一直在否定很多昨天、前天、一周前、一个月前甚至去年的当时觉得很牛逼的想法。
当然，我确实偶尔会有些商业性质的合作，但我一定是会做筛选的。我会拒绝所有可能会伤害到读者的合作，比如中转站、卖课的、黑灰类的……我都会全盘拒绝。
我会去评估写到的、推荐的每个东西，对读者来说是不是会有点用。如果没用，甚至是伤害，当然就会否掉。
找工作时如何介绍自己，打动对方，也是类似的了。
当然，你可以把上面的内容复制一下扔给你的 AI Agent，让它生成个 SKILL，甚至都不用这么麻烦，直接扔给豆包或 DeepSeek，就可以了。
