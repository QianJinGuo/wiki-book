Agent模块 用自然语言描述需求，Agent会理解任务、拆解步骤，自动完成从素材生成到视频输出的链路。有上下文记忆，后续要改背景、换风格、调节奏，直接继续说就行，不用重新描述完整背景。

### 迭代没有停止：视频 2.5 版本上线与图片4K 超高清上线

除了上面的创作者平台Pavo以外，Agnes还在继续加码。

一开始我也担心，模型API完全免费后，官方会不会放慢更新的脚步。事实证明我多虑了。继文本模型上线 1M 超长上下文后，视觉能力即将迎来了重磅升级。

图片模型 Agnes-Image-2.1-Flash 新增了 4K 超高清输出能力，支持 1K 到 4K 的自由选择，涵盖了从 1:1 到 21:9 等多种主流宽高比，这对电商展示和高清海报设计来说非常实用。

更受关注的是即将上线的 Agnes-Video-2.5-preview 视频模型。

Agnes-Video-2.5-preview这次更新，对照2.0版本主要改了几件事。

生成速度提升一倍。这个变化对做内容的人来说体感非常直接，尤其是需要多版本对比或者批量跑素材的场景，等待时间减半就是效率直接翻倍。

运镜和物理还原有改善。2.0版本偶尔会出现镜头切换突兀、人物动作在帧间漂移、物体运动不符合重力逻辑的情况。这次更新在这几个方向做了针对性优化，流畅度和物理合理性都有所提升。

新增多参考图能力。这个功能对做广告素材的人来说是刚需：同一个商品或人物在不同场景里要保持视觉一致，以前靠提示词描述控制经常不稳定，多参考图输入能直接锚定这个一致性。

字幕乱码问题也做了改善，但这类问题跟内容类型关系比较大，建议按自己的实际场景测一下，不宜提前预设效果。
