今天，我们发布 **Xiaomi-Robotics-1**，一个面向真实移动操作任务的具身基座模型。模型基于 10万 小时真实世界数据进行预训练，并结合跨本体数据完成后训练，在未见环境真实机器人任务中展现稳定的规模化收益、在复杂新任务适配和多个仿真基准中展现出强大的性能。

项目主页：https://robotics.xiaomi.com/xiaomi-robotics-1.html

# 01 — 机器人策略模型，也能进入 Scaling Law 时代吗？

过去几年，大语言模型和视觉语言模型的快速发展证明了一件事：当数据规模、模型容量和训练计算量持续提升，模型能力往往会呈现可预测的增长趋势。

但在机器人领域，这条路径并不容易复制。机器人要学习的不只是"看懂"和"说清"，还要在真实物理世界中完成动作。真实机器人数据采集依赖具体硬件、真实环境和人工遥操作，成本高、周期长、规模有限。更重要的是，传统机器人数据往往集中在少量场景和少量任务上，难以支撑策略模型获得足够强的泛化能力。

Xiaomi-Robotics-1 正是围绕这一问题展开的系统性探索。

# 02 — 10万小时真实世界轨迹，突破机器人数据瓶颈

在预训练阶段，团队使用了 10万 小时真实世界操作轨迹。这些数据通过 Universal Manipulation Interface（UMI）设备采集，覆盖家庭、商业空间、工业场景、办公室、户外等多类环境，包含大量物体交互和操作行为。

UMI 数据的优势在于不依赖特定机器人本体，同时能够更高效地采集真实世界中的操作过程。模型可以先从大规模真实操作轨迹中学习通用的动作生成表征，再在后续阶段迁移到真实机器人上。

面对 10万 小时数据，团队构建了可规模化的自动标注流程：将长轨迹切分为固定长度片段，并使用视觉语言模型对片段中的夹爪状态变化和交互物体状态变化进行描述。模型由此学习在当前视觉观察和语言条件下，生成能够推动场景状态变化的动作。该自动标注流程可在大约 2 周内完成全量 10 万小时数据的高质量标注。

# 03 — 两阶段训练：从通用动作生成，到真实机器人执行

Xiaomi-Robotics-1 采用预训练 + 后训练的两阶段训练范式。

预训练阶段：模型学习通用动作生成能力。给定当前视觉观察和语言描述，模型需要预测一段动作序列，使场景从当前状态向目标状态变化。

后训练阶段重点解决两个对齐问题：
- **本体对齐**：将预训练阶段从 UMI 数据中获得的动作生成能力，迁移到真实机器人本体上。
- **指令对齐**：将"根据状态变化描述生成动作"的能力，转化为"根据人类自然语言指令执行任务"的能力。

团队构建了约 10000 小时跨本体后训练数据，包括 7200+ 小时移动操作机器人和双臂机器人数据、1000+ 小时人工标注 UMI 数据，以及 Bridge V2、RT-1、DROID 等公开数据集。

# 04 — Scaling 效应验证

Xiaomi-Robotics-1 的核心结果之一，是在预训练阶段观察到清晰的规模化趋势。

**数据规模实验**: 使用 2.5K、5K、10K、20K 小时 UMI 数据进行训练。随着预训练数据规模增加，模型在验证集上的动作预测损失持续降低。较小数据规模下，模型更容易出现过拟合；更大数据规模下，训练过程更加稳定。

**模型规模实验**: 对比 2B、5B、10B 三个参数规模版本。随着模型规模提升，动作预测能力持续改善。

**真实机器人迁移**: 预训练阶段表现更好的模型，在后训练后的真实机器人评测中也取得更好的任务成功率。大规模预训练获得的通用表征，能够迁移到真实机器人执行中。

# 05 — 真实机器人评测

真实机器人评测覆盖鞋柜收纳、书包打包、桌面整理、沙发收拾等任务，重点考察模型在新环境、新物体条件下 out-of-the-box 的表现。

**少量数据适配复杂新任务**: 在复杂、灵巧操作任务上，模型可通过少量下游真实机器人数据（平均 <10 小时）进行高效微调，在四个测试任务中均大幅超越 Pi-0.5，体现更高的新任务学习效率。

# 06 — 仿真基准领先结果

- **RoboCasa365**: 57.4% 平均成功率，相比此前最优 46.6% 大幅提升。Composite-Unseen 任务上表现强大组合泛化能力。
- **RoboDojo**: 20.07 平均分 / 13.93% 成功率，登顶 Leaderboard，大幅刷新此前纪录（13.07分 / 8.80%）。
- **VLABench**: SoTA，平均成功率 59.1%，平均进度得分 70.3%。
- **RoboCasa**: 74.5% 平均成功率，超过 RLDX-1、Cosmos Policy、GR00T N1.6、Pi-0.5、Pi-0-FAST 等方法。

# 07 — 面向具身智能的可规模化训练路径

Xiaomi-Robotics-1 验证了一条面向具身智能的可规模化训练路径：
1. 通过大规模预训练，学习通用动作生成表征；
2. 通过跨本体机器人数据后训练，将能力迁移到真实机器人执行；
3. 通过少量下游数据微调，快速适配复杂新任务。
