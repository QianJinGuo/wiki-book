# 6.1k星AI Skill牛在哪？

李继刚 Skills 系列（四）。

前面三篇讲了输入、拆解、思考。这一篇终于到输出：表达写作类。

这组 skill 解决的是：你想明白了，怎么讲给别人听？

- **ljg-plain**：把术语和翻译腔拆掉。
- **ljg-writes**：对准一个观点下刀。
- **ljg-invest**：用"秩序创造机器"的角度看项目。

> 如果你经常写出来像 AI 说明书，可以先从 ljg-plain 开始。
