## Details
### Description
This workshop covers multi-agent orchestration, part of the [Building Agentic Systems on Amazon Web Services (AWS)](https://aws.amazon.com/marketplace/build-learn/ai-agent-learning-series?trk=416c621e-3299-473a-9e59-2abcbaa69ec5&sc_channel=em) series. Register for the workshop, then explore the [companion guide](https://aws.amazon.com/marketplace/build-learn/ai-agent-learning-series/agent-orchestration?trk=416c621e-3299-473a-9e59-2abcbaa69ec5&sc_channel=em) for additional context on the topics covered in this workshop.
You've built specialized agents. You've wired them into multi-agent systems. But agent networks without a control plane fail in predictable ways: state lost between steps, no mechanism for human sign-off on critical decisions, and silent cascading failures when one agent goes down. The root cause is the same in each case. There's no orchestration layer managing execution, state, and approval gates. This workshop shows how to build that layer using AWS services and tools from across the AI landscape, available in AWS Marketplace.
### What you'll learn
*   **Deterministic workflow orchestration with AWS Step Functions:** state management, retry logic, branching, and parallel execution across agent networks.
*   **Reasoning-driven agent coordination with Amazon Bedrock Agents:** built-in tool use, memory, and dynamic routing.
*   **Human-in-the-loop approval workflows:** gate critical decisions with approval steps that pause execution until a human signs off.
*   **Graceful failure handling with Amazon MWAA and orchestration tools:** DAG-based orchestration for complex multi-step pipelines.
