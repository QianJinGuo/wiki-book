# 腾讯研究院AI速递 20260508
生成式AI
一、马斯克官宣xAI解散，22万张GPU租给Anthropic
1.马斯克宣布xAI解散，旗下Grok及X相关业务整合进SpaceX新子部门"SpaceXAI"，此前2月SpaceX已全资收购xAI，估值1.25万亿美元；
2.SpaceX与Anthropic达成协议，将Colossus 1超22万块英伟达GPU、超300兆瓦算力供Claude使用，并探索轨道太空计算合作；
3.即日起Claude Code五小时速率限制翻倍，取消Pro/Max高峰时段削减，Opus API速率上限大幅提高。
[ https://mp.weixin.qq.com/s/NsQ1siqUVWZ04bIuB_m9Jg ](<https://mp.weixin.qq.com/s?__biz=MzA3MzI4MjgzMw==&mid=2651031726&idx=1&sn=e6eca005d102e4a279d9b460dfa0d35d&scene=21#wechat_redirect>)
二、英伟达AMD英特尔博通联手，封堵GPU算力浪费漏洞
1.OpenAI联合AMD、博通、英特尔、微软、英伟达发布开放网络协议MRC，已部署在所有前沿模型训练超算上，含甲骨文阿比林站点与微软Fairwater；
2.MRC将单次传输分散至数百条路径并行，微秒级绕开故障链路，借助SRv6源路由替代动态路由，简化网络控制面架构；
3.仅用两层交换机即可承载10万张GPU规模超算，功耗更低、负载均衡更佳，让GPU集群在故障与维护中保持协同稳定运行。
[ https://mp.weixin.qq.com/s/oIIa4M8iByVH-xpXcHSOyA ](<https://mp.weixin.qq.com/s?__biz=MjM5NTM4MTUyMg==&mid=2447915969&idx=1&sn=00f558aa245ac1e54aca2e67e04c96bb&scene=21#wechat_redirect>)
三、Claude  Dreaming学会"做梦"，一夜进化战力暴涨6倍
1.Anthropic在开发者大会推出Dreaming功能，Agent可在工作间隙读取过去最多100次会话，自动合并重复、替换过时条目、挖掘宏观规律；
2.同步发布Outcomes自动评分与多Agent编排，三件套协同将任务成功率最高提升10个百分点，文档与PPT生成效果显著优化；
3.法律AI公司Harvey接入Dreaming后任务完成率飙升约6倍，目前支持Opus 4.7与Sonnet 4.6，需申请权限。
[ https://mp.weixin.qq.com/s/9xNPYWaUw2teeSjc_kwU7A ](<https://mp.weixin.qq.com/s?__biz=MzI3MTA0MTk1MA==&mid=2652698250&idx=2&sn=55071d7c98d04b893a5907ebbfcaafb1&scene=21#wechat_redirect>)
四、DeepMind入股EVE Online，让AI研究"黑暗森林"
1.谷歌DeepMind宣布收购《EVE Online》开发商部分股权，CCP Games以1.2亿美元从Pearl Abyss赎身独立，更名Fenris Creations；
2.双方将在离线版游戏中开展受控实验，研究长远规划、记忆和持续学习，不影响在线玩家体验；
3.EVE采用单服务器宇宙架构，玩家驱动的高复杂沙盒环境为通用人工智能提供了独特的安全测试平台。
[ https://mp.weixin.qq.com/s/bxNOqT0rkEiMNTwBdeb1TQ ](<https://mp.weixin.qq.com/s?__biz=MzA3MzI4MjgzMw==&mid=2651031906&idx=1&sn=f6592d568e235765140b275ab71743ca&scene=21#wechat_redirect>)
五、豆包Seed-2.0-lite升级，  原  生  统一  全模态理解模型
1.Doubao-Seed-2.0-lite升级为豆包家族首款全模态理解模型，原生统一处理视频、图像、音频、文本，已上线火山方舟；
2.在物理、医疗等高阶推理上超越2月的Pro版本，细粒度感知与具身理解达SOTA，支持19语种语音转写，多项音频指标优于Gemini-3.1-Pro；
3.Agent、Coding、GUI能力同步升级，可承担长程任务自我拆解与跨应用操作，覆盖电竞教练、在线教育、海外电商等场景。
[ https://mp.weixin.qq.com/s/nWOSnTKD07ORQGf54LQqKQ ](<https://mp.weixin.qq.com/s?__biz=MzI0NzU1NzI5NQ==&mid=2247535894&idx=1&sn=a51109608e4e41f10dfc7a1eb6f0bfd2&scene=21#wechat_redirect>)
六、千问PC端  的  AI语音输入上线，全应用全网页可用
1.千问在PC端推出AI语音输入，Windows长按右Alt、Mac长按右Command即可唤醒，支持千问PC端及各类应用与网页中调用；
2.长按说话可自动修正口误与口水话，对周报纪要等内容进行逻辑纠错与结构化整理；
3.双击快捷键唤醒AI助手，能完成英文回信、数据查询插入、网页内容解释翻译等创作编辑任务，目前向所有用户免费开放。

1.生数科技全面开放Vidu Claw，一句话即可完成创意理解、脚本策划、视频生成到成片交付的完整流程，传统5天工作压缩至1天内完成；
