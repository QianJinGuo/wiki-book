# 一周多狂揽1000+Star！金融Agent开始进入真实工作流

**来源**: 机器之心（微信公众平台）
**日期**: 2026-07-23
**链接**: https://mp.weixin.qq.com/s/hvXU0OGU7iJ07oPUCWyUuw

## 概述

深涌智能（AlphaDojo）开源了 **DojoAgents**——一个基于 Agent Loop 的金融研究决策辅助框架。该项目发布仅一周多即获得 GitHub 1000+ Star，标志着金融 Agent 开始进入真实工作流。

## 核心功能

DojoAgents 覆盖金融研究决策的完整流程，提供以下核心能力：

- **市场扫描** — 实时监测市场动态与热点
- **新闻检索** — 基于金融事件驱动的信息获取
- **公司分析** — 多维度财务与技术面分析
- **产业链映射** — 构建上下游产业关系图谱
- **组合构建** — 根据策略自动生成投资组合
- **风险分析** — 多因子风险建模与预警
- **持仓截图诊断** — 通过截图识别和分析持仓情况

## 技术特点

- **多模型支持**: 兼容 OpenAI、Anthropic、DeepSeek、GLM（智谱）等多种大模型
- **语言**: Python
- **开源地址**: GitHub — AlphaDojo/DojoAgents（1000+ Stars）

## 意义

该项目体现了两大趋势：
1. 金融 Agent 从实验性 Demo 走向真实工作流集成
2. 国内 AI 创业公司在垂直行业 Agent 领域加速布局
