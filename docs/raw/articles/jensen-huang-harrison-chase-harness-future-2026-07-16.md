> 黄仁勋（NVIDIA CEO）与 LangChain CEO Harrison Chase 的 26 分钟对话全文。来源：YouTube（https://www.youtube.com/watch?v=Yy3JH6dDugc），CSDN 整理翻译。

## 核心观点

- **未来公司将建立在 Harness 之上**，而非传统的业务流程
- **开源权重模型已接近前沿模型性能**，但成本只有后者的十分之一
- 企业应**先使用前沿模型**了解能力天花板，再打造专业化的超级智能体
- 核心智能不能外包——"当企业需要增强自己的智能时，不能指望给第三方打个电话解决"
- **NVIDIA 工程师更愿意构建 Agent**，而非写 Python 代码
- 开放生态系统对企业至关重要：控制权、安全性、专有知识嵌入

## 为什么 NVIDIA 投资开放的 Agent 生态系统

黄仁勋认为过去六个月改变了一切——大语言模型进步、算力扩展、多模态突破。AI 终于变得实用了。

LangChain 的关键作用：将大语言模型转化为可调用的 API → RAG 系统 → 今天的 Agent 系统。AI 通过"飞轮效应"变得更强大——使用它，它变得更聪明、更实用。

虽然 Anthropic、OpenAI、Google 的工作非常出色，但企业还需要构建**专业化、特定领域、专有的 AI**。NVIDIA 要赋能这样一个世界的实现。

## 专业化：模型 vs Harness 系统

黄仁勋认为专业化首先需要足够强大的基础智能（Nemotron-3-Ultra）。但只有用 LangChain 框架和 Harness 系统将其包裹起来，使其锚定在特定领域的专业信息上，它才真正成为一个惊人的模型。

"一个聪明的人只有在能够获取关键信息时才最有用。"

甚至可以在 LangChain 的 Harness 系统中对模型进行**后期训练**（post-training），让模型更擅长利用周围的系统组件。需要一个开放的 Harness 系统让大家自行构建、应用并持续改进。

## 开源权重模型达到前沿性能

Harrison Chase 透露在 DeepAgents 中对 Nemotron-3-Ultra 的内部测试：通过优化 Harness（提示词、工具调用方式），Nemotron-3-Ultra 达到 **86% 准确率**（Claude Opus 87%，DeepSeek/MiniMax 82-83%），而成本只有 Opus 的十分之一。

黄仁勋：低成本优势在于——当智能变得廉价时，使用频率大幅增加；可以在更大的搜索空间内迭代，得到更好的答案。极快的推理速度让模型能迅速迭代并找到更优解。

## 先上前沿模型，再打造专业化智能体

黄仁勋的策略：先从前沿模型开始（了解潜力天花板），再添加子智能体/超级智能体（Super Agents），连接到专门工具上，专精于一项任务。

在 NVIDIA 内部，利用 LangChain DeepAgents 框架 + Nemotron-3 构建超级智能体，处理供应链优化、芯片设计优化等极其复杂的优化问题。

"公司本质上就是一堆超级专有的、极其重要的工作流组成的集合。"

触发专业化的信号：模型表现"足够好"了，就可以开始。很多东西可能永远不需要替换，因为前沿模型自身也在不断进步。

## 未来不是外包，而是企业内建智能

"今天的多数公司是建立在'业务流程'之上的，而未来的公司将建立在 Harness 之上。"

每家公司都建立在某种专业化的智能（知识产权/IP）之上。"将核心智能外包出去是毫无道理的。"通用技能（编程、写作）可以使用通用模型，但专业领域智能必须企业自己构建。

LangChain 将成为创建公司"操作系统"的工具。

## 跨出 AI 代码到构建智能体的范式转变

黄仁勋：NVIDIA 的软件工程师们更愿意构建智能体，而不是去写 Python 代码。"写代码就像打字，而现在他们可以减少打字，成为系统工程师，去构建和创造那些非常酷的自动化系统。"

他们在创建评估系统（Evals）、基准测试和护栏系统。将 AI 引入现实世界的工作量是巨大的，这创造了大量新的工作岗位。

## 安全与治理：AI 也需要"HR 系统"

企业部署 Agent 需要配套的安全机制和访问控制。这就像入职新员工——需要设定访问权限。AI 也需要一套类似于 HR 的系统：每个 Agent 有使命文档、工具访问权、网络/信息访问权限，IT 组织能在公司内部构建、改进和部署这些智能体。

## 缺失的拼图现已齐备

黄仁勋认为所有关键部分已经到位：世界级语言模型（Nemotron-3-Ultra）、LangChain DeepAgents 框架、快速上手的蓝图（Blueprints）、确保系统安全的 NIM 运行环境。全球每家公司、每个开发者现在都能创建这些超级智能体，并部署在任何地方——云端、本地。

"没有任何理由不去拥抱它。"
