# 智谱GLM-5.2上线华为云，可通过多款产品体验

6月17日，智谱正式发布新一代开源旗舰模型GLM-5.2，当前已上线华为云，并成功对接多款产品。GLM-5.2是智谱迄今能力最强的开源模型，专为长程任务能力而生。同时，其Coding能力全面增强，为开发者带来更强体感、更高可用性的智能开发体验。在Artificial Analysis综合榜单中，GLM-5.2位居全球第四，达到开源模型SOTA水平；在全球百万用户参与盲测的前端开发评估系统Code Arena中，GLM-5.2斩获全球可用模型第一。

  


  
  
  
  


  


在模型的长程任务上, GLM-5.2可以自主完成任务拆解、架构设计、前后端开发、接口联调、测试修复与打包部署，最终交付一个真正可部署、可上线的多端应用，覆盖Web、移动端与小程序。这条从需求描述到多端应用部署的完整链路，过去往往需要一支团队协作数周，如今GLM-5.2能在一次连续的长程任务中跑完。在FrontierSWE、SWE-Marathon、PostTrainBench等多个长程任务基准测试中，GLM-5.2表现优异，成为排名最高的开源模型。

  


在Coding能力上，GLM-5.2在前端、后端、长程任务等开发场景下的成功率实现长足提升，复杂系统工程与深度调试更稳，实现更强体感，更实用的Coding能力。

  


华为云从算力调度、网络通信、底层算子三个维度进行了全栈软硬协同优化，显著提升了GLM-5.2的推理效率。 

  


  
  


**在算力调度层，** 针对MoE架构在昇腾平台上实现了Layer级的动态均衡调度，解决了传统MoE路由不均导致的计算长尾效应；

  


**在通信层，** 依托华为云CloudMatrix智算云服务高速互联架构，利用零拷贝技术消除了节点间的通信开销，实现了千卡规模的高效协同；

  


**在算子层，** 结合昇腾硬件特性，对Attention及Linear等高频基础算子进行了深度融合与重构，大幅降低了推理时延。

  


华为云积极拥抱主流开源SOTA模型，提供多样化业务接入方式，保障企业级服务可用性，加速企业AI创新落地。目前华为云MaaS模型即服务平台已为开发者提供免部署、一键调用智谱GLM-5.2 API的Tokens服务，支持在线体验。华为云码道CodeArts代码智能体、华为云智果AgentArts企业级智能体平台也已完成与 GLM-5.2 的深度适配，进一步降低AI开发门槛，提升智能体创新效率与任务执行能力。
