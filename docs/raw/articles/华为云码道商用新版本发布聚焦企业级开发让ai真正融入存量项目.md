# 华为云码道商用新版本发布，聚焦企业级开发让AI真正融入存量项目

近日，华为云码道（CodeArts）代码智能体新版正式发布，围绕企业研发中的存量项目增量开发、企业级安全等高频场景实现多项能力升级, 推动AI更加深度融入企业研发全流程。自5月30日正式商用以来，华为云码道持续围绕企业研发场景快速迭代，不断增强企业级开发能力。

  


  
  
  
  


  


作为华为云智能体平台的核心原子能力，华为云码道不仅是一款AI编程工具，更是连接人类（碳基）世界与硅基世界的“翻译器”。它向下连接模型、算力、知识库、工具链等“硅基黑土地”能力，向上连接开发者与智能体，通过自然语言理解、任务规划、自主执行和工具协同等能力，让开发者能够以自然语言完成需求分析、方案设计、代码生成、调试验证等研发工作，推动软件研发从“AI辅助编码”迈向“智能体协同开发”。

  


新版本重点增强了存量项目增量开发能力，实现云端CodeBase多端支持，IDE客户端、VSCode插件及CLI/TUI统一接入云端CodeBase，可支持千万行级代码仓统一索引与检索，提升复杂代码仓理解能力，在存量项目开发过程中实现更加精准的代码生成，并有效降低Token消耗。

  


同时，华为云码道进一步扩充专家Skill市场，新增覆盖单元测试、代码评审、文档生成、安全检查、编译构建、代码重构等多个开发场景的专家Skills，为开发者提供开箱即用的专业能力，进一步提升研发效率。

  


针对企业开发环境，华为云码道CLI/TUI工具新增Linux平台支持，在Linux服务器开发环境中功能完整可用，与Windows、Mac平台体验一致，安装流程简单快捷。支持网络访问控制、命令白名单、路径访问限制等多项安全策略，为AI安全执行代码提供保障。

  


在企业级能力方面，新版本新增安全隔离代码仓功能，企业可将核心代码仓设置为安全隔离仓，AI能力将无法访问相关仓库，进一步保障企业核心代码安全。同时支持基于IAM的SSO单点登录，以及公网IP白名单配置，帮助企业统一身份认证、加强访问控制，满足数据安全与合规管理需求。

  


随着AI编程进入企业研发深水区，企业关注的重点正从代码生成延伸到复杂代码理解、研发协同和安全治理。此次版本升级，进一步完善了华为云码道在企业级研发场景下的能力体系，让AI更深入参与企业软件开发流程，助力企业研发迈向更加智能、高效的新阶段。
