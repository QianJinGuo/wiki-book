# 使用营销策略助手--数据智能体-火山引擎

营销策略助手是一款基于 AI 深度思考与大数据分析的数字营销中枢，通过 "目标输入 - 策略生成 - 任务配置 - 动态优化" 的智能闭环，实现从营销要素解析、智能圈群匹配到全渠道策略输出的全流程自动化，为企业打造具备持续进化能力的 "AI 总参谋" 系统。

## 智能生成营销策略

### 输入营销目标

**触达活动描述**：根据实际需求输入活动背景、主题、内容、面向人群以及活动目标等描述信息。点击「换一换」将自动生成营销目标描述。

**触达时机**：如果在实际触达活动里需要指定触达时机则可以设置该字段。设置「触达时段」后，每个用户在实际下发时，系统将会在该触达时段范围内寻找一个最佳触达时机；如果未设置「触达时段」，系统将默认筛选最近 7 天的时间范围。

**触达内容**：设置「触达通道及通道物料」内容后，每个用户在生成营销方案及实际触达时，系统将会使用该指定的内容进行生成和消息触达。如果未设置，系统将自动生成营销内容。

**触达通道**：可以选择多个通道，每个用户在时机下发时，系统会基于历史营销数据表现，确定最佳的触达通道和通道物料。可选择短信或Webhook。

**指定通道物料**：选模板、选物料、选权益。

## 智能圈选目标人群

1. **智能体深度思考**：根据输入的触达活动描述，推理业务意图、拆解目标，分步骤获取行业实践经验、画像标签识别以及客群的创建。
2. **智能圈选并生成目标人群**：
   - 对行业客群选择的实践经验和客群画像特征进行总结，帮助精准定位目标客群。
   - 自动构建客群，提供客群id、客群名称、客群数量、客群描述等信息。
3. **分群详情**：查看详细的分群圈选规则，可按实际业务使用对分群圈选条件进一步进行修改。
4. **分群洞察**：根据用户标签属性的数据特征提供更深入的洞察结果。

## 智能生成策略

### 方案推荐

点击「生成营销方案」，系统将会生成分群的多种拆分方案，以便用于精准营销。可根据业务场景选择合适的营销方案。

### 策略生成

点击「应用方案」，将基于所选的营销方案生成详细的营销触达策略，包含触达时机设计、触达通道及内容设计。

### 触达任务配置

点击「编辑触达任务」，基于策略自动完成触达任务配置，也可二次编辑修改配置。

- **受众用户**
- **触发条件**
- **触达配置**

支持存为草稿和提交审批。

## 营销任务管理

可查看历史创建的营销任务，支持查看任务详情、查看审批进度、复制、编辑、删除等操作，也可新增触达任务。

- **查看任务配置**：查看每个任务的配置信息，包括受众用户、触发条件、触达配置等
- **编辑任务配置**：可修改每个任务的配置信息
- **查看数据分析结果**：查看任务的数据效果

## 个性化触达

基于用户数据和历史营销数据，为不同用户匹配个性化的营销内容，自动匹配个性化触达时机、物料、权益、模板。
