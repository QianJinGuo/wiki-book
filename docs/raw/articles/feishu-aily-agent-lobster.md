# 飞书aily：企业级Agent智能伙伴
## 核心产品
飞书春季发布会推出升级版**aily**（飞书龙虾），以Agent形态成为飞书联系人，支持 Skills 人设配置。
## 核心功能
### AI编辑部（6角色协作）
| 角色 | 职责 |
|------|------|
| 总编虾 | 深度分析选题，输出文章提纲 |
| 记者虾 | 搜集选题相关资料 |
| 编辑虾 | 根据资料撰写初稿 |
| 校对虾 | 校对文章内容 |
| 排版虾 | 对文章进行排版 |
| 运营虾 | 将文章发布到网站 |
### Skills配置示例（总编虾）
```
你是【总编虾】，AI编辑部的总负责人，只做总编的本职工作，禁止越权做其他角色的任务。
【核心职责】根据选题深度分析受众、核心看点、传播逻辑，输出专业、可落地的文章提纲。
【硬性规则】
1. 提纲必须包含：选题核心立意、受众画像、全文结构（至少3大章节×2小节），每小节标注「需要记者搜集哪类资料」
2. 写完提纲后，必须调用feishu-bitable技能，把提纲同步到「AI编辑部进度看板」
3. 只能使用feishu-bitable、feishu-docs这2个技能
4. 完成所有工作后，明确告知用户「总编虾任务已完成」
```
### 企业级长任务（aily专业版）
单一任务包含：
- 月度经营汇报PPT（核心指标复盘、业务趋势分析、项目进度跟踪、问题风险复盘、下月工作计划）
- 配套数据可视化图表
- 多项目进度跟踪看板
### 飞书妙搭Agent
一句话Prompt即可创建完整应用（如狼人杀游戏），包含概要设计→开发计划→自动执行→一键发布。
### 多维表格Agent
自然语言搭建系统：选模式（轻量/标准/专业）→ 自动系统设计 → 一键搭建 → 等待完成。支持视图（看板/列表/甘特图）、数据仪表盘、自动化提醒、权限设置。
## 企业级优势
- **安全可控**：操作权限与用户本人一致，全链路留痕，敏感操作人工确认
- **上下文理解**：基于飞书消息、文档、表格的全场景业务上下文
- **全链路闭环**：从需求到结果，Agent不止输出方案更能直接执行
## 产品体系
- **aily**：智能伙伴，Agent形态联系人
- **飞书妙搭Agent**：Code Agent，一句话创建应用
- **多维表格Agent**：自然语言搭建表格系统
飞书已完成从办公协作平台到Agent时代智能操作系统的关键跃迁。
