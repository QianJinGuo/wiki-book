# 世界模型的DeepSeek时刻！魔芯Flash World Model降本70%，跑出50FPS实时交互

机器之心发布  


今日，专注 4D 世界模型领域研究和产业化的魔芯科技正式发布 MoWorld，一个 Flash World Model—— 全球首个超高帧率的交互式世界模型。它无需依赖高端 GPU，即可实现影视级画质、最高 50 FPS 实时交互，并以更低成本推动世界模型走向大规模应用。

  


速度对于世界模型驱动的各类决策系统至关重要，不管是实时机器人控制还是导航，都需要世界模型高效响应，目前主流的世界模型帧率在 30 帧以下；如何进一步提高帧率，服务实时决策系统变得至关重要。

  


魔芯科技团队首次提出了 Flash World Model 的概念 —— 高帧率才能保持世界模型在真实场景中可用。在魔芯和其战略股东华为团队共同的极致优化下，MoWorld 首次在纯国产 Ascend NPU 平台上打通了世界模型从预训练、自回归蒸馏到实时推理部署的完整技术链路。依托这一全栈优化体系，14B 参数 MoE 的 MoWorld 在国产 NPU 超节点上最高可实现 50 FPS 实时推理，支持长时程、可交互的视频世界生成，首次将世界模型带入国产化、低成本且高帧率实时交互的时代。

  


MoWorld 证明了国产 NPU 能够完成世界模型训练，并有承载大规模世界模型实时部署的能力，让中国芯真正可以快速预测世界、模拟世界。相比 GPU，经过 NPU 优化后的 MoWorld 的推理成本降低 70%，大幅降低了世界模型的部署门槛，让高帧率交互式世界模型真正具备了规模化落地的可能，也让行业看到了属于国产世界模型的「DeepSeek 时刻」。

MoWorld：数据、训练和系统优化实现 Flash World Model

  


在 MoWorld 低成本与优秀效果的背后，是基于魔芯科技对国产世界模型持续不断的探索所构建的全新技术架构。魔芯推出了先进的数据引擎、预训练、蒸馏优化与推理体系架构。

  


一、行业领先的数据引擎：高质量世界建模资产拉开预训练效果代际差

  


与传统视频生成模型不同，世界模型训练不仅需要视频和文本，还需要精确的相机轨迹与空间信息，以实现「4D 感知」。

  


为此，MoWorld 基于过去在 3D/4D 重建领域的相关成果，构建了可扩展的数据生产引擎和自动化数据治理管线，将海量原始视频高质量转换为世界模型训练数据。通过 3D 几何一致性、轨迹精度、多视图稳定性等多维质量筛选，以及几何补全、动态区域过滤和轨迹验证等关键流程，从海量数据中筛选出高质量训练资产，为 MoWorld 的低成本、高性能训练提供了坚实的数据基础。

  


  


二、高性能低成本的系统闭环：预训练、蒸馏与推理的三阶段协同重构

  


针对国产 NPU 集群高并行、长序列计算能力强的特点，MoWorld 在预训练阶段设计了超密集注意力并行和输入 Token 并行两项关键技术，将注意力计算和超长视频序列拆分到多台 NPU 协同执行，在有效控制显存占用的同时，大幅提升了训练效率，支撑大规模世界模型完成长时程视频训练，训练得到长达 2 分钟的基础模型。

  


在蒸馏阶段，MoWorld 压缩到了 4 步实时推理的自回归学生模型，并引入结合全局锚点和相机稳定一致性的 Memory 机制保留历史关键信息。同时，创新性地跳过传统蒸馏中计算开销巨大的教师轨迹采样初始化过程，显著降低蒸馏成本，为世界模型的低成本训练和实时部署提供了关键支撑。

  


三、混合精度动态量化并行推理，14B MoE 世界大模型实现最高 50 FPS 交互生成。

  


MoWorld 将实时推理问题分解为三个层级协同优化：流水线层、并行层与算子层，在资源受限场景实现显存高效利用，容纳庞大的模型参数、通信缓冲与推理状态；同时压缩端到端时延，将用户控制信号实时反映到生成帧中，并以流式方式持续输出。

  


具体来说，MoWorld 首先在整体流程上尽可能减少无效开销，只在初始阶段完成一次条件编码，并在后续生成过程中复用已有结果，把更多显存留给核心生成模块；随后在多 NPU 部署上，将解码与主干生成解耦，并把最重的计算任务分散到多张 NPU 协同执行，以降低单设备压力、缩短整体时延；在底层执行上，再通过混合精度量化与高效注意力计算进一步减少显存搬运与冗余计算。三层协同的结果，是 MoWorld 不只是把大模型「跑起来」，而是把 14B 参数世界模型真正推进到可实时交互的部署区间。

  


  


基于上述三层协同设计，MoWorld 使 14B 参数的 MoE 世界模型能够在华为昇腾 NPU 平台上实现最高 50 FPS 的实时推理。在保持大模型生成质量与控制能力的同时，这套系统也为低显存、低时延条件下的 Flash World Model 部署提供了完整的工程实现路径。

  
MoWorld 打开了世界模型产业落地的新局面

  


MoWorld 的价值并不止于模型能力验证，更在于其已经呈现出清晰的产业化落地路径。依托可控相机运动、长时程一致性与实时交互能力，MoWorld 正在向影视制作、互动内容生产、数字场景资产化以及空间智能等方向延展，逐步从技术系统演进为可直接赋能业务场景的基础能力平台。

  


以影视制作为例，剧情动作与相机运镜往往需要同时成立，而现有主流视频生成模型虽然能够较好完成动作生成，却难以稳定控制镜头运动。针对这一痛点，MoWorld 提供了「相机 - 剧情联合控制」的应用路径：先生成具备稳定相机路径的场景视频，进一步重建对应的 3DGS 场景，再将该场景在目标运镜下的渲染结果作为参考输入下游视频模型，使创作者能够将重心集中在角色动作、叙事事件与镜头语言本身，而显著减少背景漂移、视角跳变和场景身份不一致等问题。这意味着，MoWorld 不只是提升了生成质量，更开始以底层空间能力的形式进入商业化内容生产流程。而对于具身仿真和自动驾驶，这类高帧率的 Flash World Model 的作用更大 —— 能够让这些自动系统以前所未有的速度进行决策。

  


世界模型的竞争正在从「谁生成得更像」，转向「谁能在真实算力约束下更稳定、更实时、更经济地运行」。未来真正有价值的世界模型，不仅要具备高质量生成能力，更要具备长时程稳定性、实时交互能力、系统部署能力和成本效率。

  


魔芯团队自创立之初便笃定空间智能与物理 AI 赛道，依托多年落地积累，团队沉淀海量真实三维物体、室内场景、动态交互模型资产，形成行业稀缺的原生 3D 数据壁垒。魔芯科技创始人陈天润师从潘云鹤院士，坚守院士开创的 3D 视觉知识技术路线。潘院士预判，未来全球物理 AI 将全面迈入 4D 动态世界模型时代，时序、物理、交互将成为通用 AI 必备底层能力。未来 3-5 年，魔芯 4D 世界模型将冲刺全球通用世界模型基模第一梯队，打造国产自主世界模型标杆，代表中国空间 AI 技术与国际顶尖团队正面竞争。

结语

  


MoWorld 作者包括魔芯科技团队和浙江大学潘云鹤院士团队等。成果标志着魔芯科技在世界模型方向上完成了一次关键跨越：从单点模型能力验证，走向端到端实时交互系统构建；从依赖通用 GPU 的传统范式，走向基于国产 NPU 的全栈原生重构；从「生成视频」，走向「生成并模拟一个可持续演化的世界」。更重要的是，MoWorld 将世界模型首次带入高帧率实时交互时代，让世界模型真正具备了持续感知、持续生成与持续交互的能力。

  


另外，MoWorld 不是简单把世界模型迁移到国产芯片上，而是在全 NPU 生态上原生构建了一条可规模化延展的世界模型技术路径；它不仅证明了国产算力能够训练世界模型，更证明了国产算力能够支撑高帧率、低成本、实时交互的世界模型规模化部署。这也是国产算力第一次在实时交互世界模型领域，给出了一套完整且清晰的系统答案。

  


对整个行业而言，MoWorld 的发布释放出的信号也非常明确。世界模型的竞争正在从单一的生成质量比拼，转向高帧率交互、系统能力、长时程稳定性与成本效率的综合竞争。未来，真正具有竞争力的世界模型，不仅要生成得更逼真，更要运行得更流畅、交互得更实时、部署得更经济。谁能在真实算力约束下打通训练、蒸馏、推理和部署全链路，谁就更有机会将世界模型从实验室带向空间智能、具身智能、数字孪生、影视预演和交互内容生产等真实场景。

  


据悉，MoWorld 背后的魔芯科技凭借其独特的模型能力和突出产业化进展，于近期获得国家级战略储备资本、中东知名美元机构、头部市场化基金以及十余家产业资本共计过亿美元融资，其中包括 B 站旗下干杯基金、传音股份旗下星睿基金等；在此之前，魔芯科技已获得华为旗下哈勃投资、联想控股旗下基金投资。魔芯科技正和产业伙伴一起，开拓世界模型赋能创意实现的新可能。
