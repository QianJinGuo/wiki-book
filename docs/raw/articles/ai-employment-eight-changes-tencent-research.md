# 我们刚过了人类最后一个劳动节？AI新职业的八个变化
> 腾讯研究院吴朋阳 | 2026-05-11
## 核心发现
1. **岗位总量倍增**：7家公司在招岗位 718→1570，8个月翻倍
2. **人力从研发转向商业化**：使能者占比↓9pp，协作者+推广者↑11pp
3. **使能者内部换血**：推理/研究岗位增长，预训练/数据岗位减少
4. **部署军队成型**：FDE从零星→100+岗位，OpenAI占78.5%
5. **推广者激增**：66→275（4x），客户成功/合作伙伴销售爆发
6. **支持者占比稳定**：AI增强而非替代白领岗位
7. **治理者分化**：Anthropic+OpenAI贡献90%，多数公司投入不足
8. **中美差异**：大方向一致，中国使能者更高/推广者更低/治理者更少
## 关键洞察
- **API key 卖不动传统大企业** — 一个API key解决不了银行的合规流程或医院的电子病历对接 → 模型公司必须配部署军队
- **"自己已经被AI改造过的人"** — OpenAI ADE要求"你必须是AI编程工具的深度用户，已经深度定制了自己的开发工作流"
- 对软件工程师：过去转型方向"做云/微服务"，未来"走到客户面前，把AI变成生产力"
- 对白领：岗位还在，但"能用 AI 重新定义自己工作方式"的人才有优势
- 部署岗三层递进：FDE（驻场工程交付）→ ADE（产品采用指导）→ ADM（方法论标准化）
## 五类岗位框架（ECPGS）
| 类别 | 定义 | 典型岗位 |
|------|------|----------|
| 使能者 Enablers | AI技术研发 | 推理工程师、研究科学家、算力工程师 |
| 协作者 Collaborators | 产品化和交付 | FDE、ADE、解决方案架构师 |
| 推广者 Promoters | 市场营销和销售 | 客户经理、合作伙伴销售、客户成功 |
| 治理者 Governors | 安全合规治理 | 安全工程、AI合规官、威胁调查 |
| 支持者 Supporters | 后勤运营 | HR、财务、法务、IT运营 |
