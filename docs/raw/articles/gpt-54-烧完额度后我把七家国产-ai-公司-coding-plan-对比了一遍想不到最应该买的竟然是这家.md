# GPT-5.4 烧完额度后，我把七家国产 AI 公司 Coding Plan 对比了一遍，想不到最应该买的竟然是这家！
> AI 时代，有两种行为：
>
> 一种，活在别人的评测里，把模型的强当自己的强，痴人说梦；
>
> 另一种，活在真实的实战里，用最顶级的 AI，武装自己。
>
> 前者在噪音里坐享"技术平权"，后者在疼痛中完成"自我进化"。
朋友们好，我是行小招。
这两天用 OpenClaw 跑 GPT-5.4 搞任务，效果确实猛，但 token 烧得更猛，plus 账号的周额度一两天就被干完了，额度像沙漏一样往下掉，心跟着一起疼，难不成继续买个 plus？
所以我开始认真研究国产 Coding Plan，把阿里、字节、腾讯、百度、MiniMax、月之暗面、智谱七家全扒了一遍，整了一张对比表。扒完第一反应：
四大云厂商，你们是商量好的吧？
阿里百炼、字节火山方舟、腾讯云、百度千帆，Lite 统一 40 元/月，Pro 统一 200 元/月，首月优惠阿里腾讯百度都是 7.9 元，字节 9.9 元。限额全是每 5 小时 1200 次请求、每周 9000 次、每月 18000 次，连数字都一模一样。
支持的模型也高度重叠，Kimi-K2.5、MiniMax-M2.5、GLM 系列，你在这家能用的，换一家基本也有。
**四大云厂商的 Coding Plan，像是同一个产品经理写的 PRD**
独立模型厂商画风完全不同，拉个表看得更清楚：
厂商  |  最低月费  |  计量方式  |  首月优惠  |  核心差异
---|---|---|---|---
阿里/腾讯/百度  |  ¥40  |  API 请求次数  |  ¥7.9  |  模型多，最便宜
字节火山方舟  |  ¥40  |  API 请求次数  |  ¥9.9  |  附带 ArkClaw 权益
MiniMax  |  ¥29  |  Prompt 次数  |  ¥9.9  |  无周限额，连续开发友好
月之暗面 Kimi  |  ¥49  |  Token  |  无  |  官方模型体验完整
智谱 GLM  |  ¥49  |  Prompt 次数  |  无  |  GLM-5 强，但高峰 3 倍扣量
计量方式这列才是重点。三种完全不同的算法：API 请求次数（1 次提问消耗 5~30 次请求）、Prompt 次数（1 次提问 = 1 次）、Token 计量（按输入输出长度算），直接比数字毫无意义。
MiniMax 的亮点是没有周限额，只有 5 小时滑动窗口，不会出现"周三额度见底周末干瞪眼"的窘境。智谱 GLM-5 有个坑：高峰期下午 2-6 点额度消耗翻 3 倍，你以为买了 400 prompts/5h，高峰期实际只有 130 次出头。
但数字只是表面。
**同一个模型在不同平台上跑出来的体感，真的不一样**
这是我自己踩出来的，我在百度千帆上用 Kimi K2.5，发现它竟然不支持图片识别。Kimi K2.5 是个多模态模型，图片理解是核心能力之一，结果上了云平台被砍了？
云厂商部署开源模型，参数设置、推理配置、功能裁剪都是自己定的，你以为用的是同一个模型，拿到手可能是个阉割版。所以如果认准了某个模型，直接去官方，贵一点，但功能完整，不会被偷偷砍能力。
回到怎么选。
薅羊毛阶段，7.9 元的阿里、腾讯、百度加上 9.9 元的字节，四家挨个开一遍，加起来不到 40 块，能用好几个月，顺便对比哪家部署的模型最对你胃口。
长期选一个的话，MiniMax 29 元的 Starter 是目前最便宜的月付方案，无周限额适合连续开发，但模型能力天花板在那摆着，复杂任务可能吃力。
对某个模型有执念的，直接去官方，Kimi 去月之暗面，GLM 去智谱。
我自己的策略：先薅完四大云厂商的首月羊毛，重点体验 MiniMax M2.5 和 Kimi K2.5 的官方版本。
等薅完了再定长期方案，反正这些厂商的政策变化比天气还快，三个月后可能又是另一番景象。
