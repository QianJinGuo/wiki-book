# Qoder 企业版全球发布：让 AI Coding 从“个人工具”长出“组织能力”
今天，Qoder 全球站正式发布「企业版」，Qoder CN 和 Qoder 的企业版在国内、国际阿里云云市场同步开售。  
这不是一次简单的功能升级、这次发布，覆盖从 Credits 资源池化、按人按库的模型分发，到企业专属 Plugin/Skill 市场、十万级大库知识引擎，再到五层纵深防御安全体系，让 AI Coding 从“个人研发工具”长成“企业组织能力”。
  
  
**共享 Credits 资源池，企业用量灵活分配**
企业规模化使用 AI 编码工具，关心的是：资源怎么在团队间灵活调配？成员变动如何快速响应？预算和续费怎么集中可控？
  
  
Qoder 企业版收费采用 Credits 完全资源池化——企业以共享资源包的方式持有 Credits，管理员可以按需为成员或计费组动态分配额度，让资源在团队间灵活流转。云市场兑换码同步支持席位的新增、调整与续费，管理员可在云市场后台随团队变化即时操作，订阅管理更灵活、更轻量。  
  
**模型权限，企业按人、按代码库灵活分配**
研发团队的角色千差万别——核心算法工程师、业务开发、外包、合作方，他们能用的模型会有些区别。Qoder 企业版把模型开关交到企业手里，企业管理员可按群组配置可用模型范围——内置模型可分组屏蔽/开启，模型可按人、代码库开放和下发，还支持导入+
